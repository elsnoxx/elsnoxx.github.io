Výrazy

  • 5.9/7
    • Tento výraz je validní, protože provádí běžné dělení dvou desetinných čísel.
  • (floor 5.9)/7
    • Tento výraz je validní, protože funkce floor zaokrouhlí číslo 5.9 na celé číslo (5) a následně provádí dělení.
  • floor 5.9/7
    • Tento výraz je také validní, protože Haskell automaticky použije funkci floor na číslo 5.9 a následně provede dělení.
  • fromIntegral floor 5.9/7
    • Tento výraz není validní, protože chybí závorky kolem fromIntegral a floor. Měl by být napsán jako fromIntegral (floor 5.9) / 7.
  • fromIntegral (floor 5.9)/7
    • Tento výraz je validní, protože nejprve aplikuje funkci floor na 5.9, pak převede výsledek na celé číslo pomocí fromIntegral a nakonec provádí dělení.
  • div (floor 5.9) 7
    • Tento výraz je validní, protože používá funkci div k celočíselnému dělení výsledku funkce floor a čísla 7.
  • (floor 5.9) div 7
    • Tento výraz je také validní, protože div může být použito jako infixová operace v závorkách.
  • (floor 5.9) `div` 7
    • Tento výraz je validní a používá zpětné uvozovky kolem div, což je možný způsob zápisu infixových funkcí v Haskellu.
  • mod 10/2 3
    • Tento výraz není validní, protože použití mod vyžaduje dva argumenty v závorkách. Správný zápis by byl mod (10/2) 3.
  • mod (floor (10/2)) 3
    • Tento výraz je validní, protože nejprve aplikuje funkci floor na výraz 10/2 a následně použije funkci mod pro výpočet zbytku po dělení výsledku tímto číslem 3.

Faktorial

První možnost

factorial :: Int -> Int
factorial 0 = 1
factorial n = n * factorial (n - 1)

Druhá možnost

factorial' :: Int -> Int
factorial' n | n == 0 = 1
         | otherwise = n * factorial' (n - 1)

Třetí možnost

factorial'' :: Int -> Int
factorial'' n = if n==0 then 1 else n * factorial'' (n - 1)

Fibonačiho číslo

První možnost

fib :: Int -> Int
fib 0 = 0
fib 1 = 1
fib n = fib (n - 1) + fib (n - 2)

Druhá možnost

fib' :: Int -> Int
fib' n | n==0 = 0
    | n==1 = 1
    | otherwise = fib' (n - 1) + fib' (n - 2)

Třetí možnost

fib'' :: Int -> Int
fib'' n = tmp n 1 1 where
    tmp 0 a _ = a
    tmp x a b = tmp (x - 1) b (a + b)

Přestupný rok

První možnost

leapYear :: Int -> Bool
leapYear x = x `mod` 4 == 0 && x `mod` 100 /= 0 || x `mod` 400 == 0

Druhá možnost

leapYear' :: Int -> Bool
leapYear' x | x `mod` 400 == 0 = True
        | x `mod` 100 == 0 = False
        | otherwise = x `mod` 4 == 0

Maximum

Maximum ze dvou čísel

max2 :: Int -> Int -> Int
max2 a b = if a >= b then a else b

Maximum ze tří čísel

max3 :: Int -> Int -> Int -> Int
max3 x y z = (x `max2` y) `max2` z

combinations :: Int -> Int -> Int
combinations n k = factorial n `div` (factorial k * factorial (n-k))

Maximum ze tří čísel

combinations' :: Int -> Int -> Int
combinations' n k = fromIntegral(factorial n) `div` fromIntegral(factorial k * factorial (n-k))

Primární číslo

isPrime :: Int -> Bool
isPrime 1 = True
isPrime n = isPrimeTest n (n - 1) where
isPrimeTest n 1 = True
isPrimeTest n i | n `mod` i == 0 = False
        | otherwise = isPrimeTest n (i - 1)

Průměr třech čísel

howManyAboveAverage a b c = if aboveAvg a + aboveAvg b + aboveAvg c > 0 then 1 else 0
    where
       aboveAvg x = if x > averageThree a b c then 1 else 0
       averageThree a b c = fromIntegral (a + b + c) / 3

length' [ ] = 0
length'(_:xs) = 1 + length' xs

sumIt :: [Int] -> Int

sumIt(x:xs) = x + sumIt xs

getTail :: [a] -> [a]

getHead :: [a] -> a
getHead (x : _) = x

Posledni prvek z listu

getLast :: [a] -> a
getLast [x] = x
getLast (_ : xs) = getLast xs

Je prvek v listu

isElement :: Eq a => a -> [a] -> Bool
isElement _ [ ] = False
isElement a (x : xs) | a == x = True
            | otherwise = isElement a xs

Napis funkci ktera vybere element s nejvyssi hodnotou

max'' :: [Int] -> Int
max' (x : xs) = tmp x xs where
        tmp m [ ] = m
        tmp m (x : xs) | m < x = tmp x xs
            | otherwise = tmp m xs



max'' :: [Int] -> Int
max'' [x] = x
max'' (x : y : rest)
          | x < y = max'' (y : rest)
          | otherwise = max'' (x : rest)

Napis funkci ktera prehaze prvky v seznamu

reverse' :: [a] -> [a]
reverse' [ ] = [ ]
reverse'(x : xs) = reverse' xs ++ [x]
reverse'' :: [a] -> [a]
reverse'' x = tmp x [ ] where
          tmp [ ] ys = ys
          tmp (x : xs) ys = tmp xs (x : ys)

Napiste funkcinasobeni skalarem

scalar :: [Int] -> [Int] -> Int
scalar [ ] [ ] = 0
scalar (x : xs) (y : ys) = x * y + scalar xs ys

Odstraní všechny výskyty nuly v seznamu

nonZeros :: [Int] -> [Int]
nonZeros [ ] = [ ]
nonZeros (x:xs)
          | x == 0 = nonZeros xs
          | otherwise = x : nonZeros xs

Rotaci seznamu doleva o jeden prvek.

rotateLeft1 :: [a] -> [a]
rotateLeft1 [ ] = [ ]
rotateLeft1 (x:xs) = xs ++ [x]

Rotaci seznamu doprava o jeden prvek.

rotateRight1 :: [a] -> [a]
rotateRight1 [ ] = [ ]
rotateRight1 xs = last xs : init xs

          | otherwise = x : oddMembers xs
