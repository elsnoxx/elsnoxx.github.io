Algoritmy a Seznamy

V této lekci se naučíme, co jsou to algoritmy a jak fungují seznamy! K vytváření projektů budeme používat šikovné prostředí od Microsoftu s názvem Visual Studio Code. Potřebujete pomoc? Návod na stažení a zprovoznění najdete zde.

Opakování

Tento test je zaměřen na procvičení základních a středně pokročilých konceptů v Pythonu. Ověříte si své dovednosti v oblasti generování náhodných čísel, měření času a manipulace s textovými barvami. Po dokončení testu budete mít možnost zkontrolovat své odpovědi a získat zpětnou vazbu.

Co jsou to algoritmy?

Algoritmy jsou jako recepty v kuchařce – přesně říkají, co máme dělat krok za krokem. Více se dozvíte zde. 🍳

Seznamy a indexy

Seznam je jako dlouhý řetěz. Každý dílek (neboli prvek) má svoje místo, kterému říkáme index. 📍

Program pro třídění

V této sekci si ukážeme, jak vytvořit program, který umí seřadit čísla! Postup je jednoduchý, rozdělíme si práci na několik částí.

🛠️ 1. Ukázka záměny hodnot

V souboru lekce2_zamena.py máme jednoduchou ukázku, jak mezi sebou zaměnit hodnoty dvou proměnných: 

    Ukázka kódu:
    x = 5
    y = 10
    
    # Zaměna hodnot
    temp = x
    x = y
    y = temp
    
    print("Hodnoty po záměně:", x, y)

📥 2. Vstup od uživatele

Nejprve připravíme část programu, kde uživatel zadává čísla. Použijeme nekonečný cyklus, který se ukončí jen, pokud vstup splňuje pravidla: 

    Ukázka kódu:
    my_list = []
    
    while True:
        user_input = input("Zadej číslo (nebo 'q' pro ukončení): ")
        if user_input.isnumeric():
            my_list.append(int(user_input))
        elif user_input.lower() == 'q':
            break
        else:
            print("Neplatný vstup, zkus to znovu.")
    
    print("Seznam před tříděním:", my_list)

🔄 3. Cyklus algoritmu

Nyní vytvoříme algoritmus Bubblesort, který seřadí seznam: 

    Ukázka kódu:
    def bubble_sort(sequence):
        exchange_happened = True
        list_length = len(sequence)
    
        while exchange_happened:
            exchange_happened = False
            for i in range(list_length - 1):
                if sequence[i] > sequence[i + 1]:
                    # Záměna hodnot
                    sequence[i], sequence[i + 1] = sequence[i + 1], sequence[i]
                    exchange_happened = True
        return sequence
    
    sorted_list = bubble_sort(my_list)
    print("Seznam po třídění:", sorted_list)

📊 4. Statistika třídění

Na závěr můžeme přidat statistiku, například počet iterací: 

    Ukázka kódu:
    def bubble_sort_with_stats(sequence):
        exchange_happened = True
        list_length = len(sequence)
        iterations = 0
    
        while exchange_happened:
            exchange_happened = False
            iterations += 1
            for i in range(list_length - 1):
                if sequence[i] > sequence[i + 1]:
                    sequence[i], sequence[i + 1] = sequence[i + 1], sequence[i]
                    exchange_happened = True
        return sequence, iterations
    
    sorted_list, total_iterations = bubble_sort_with_stats(my_list)
    print(f"Seznam po třídění: {sorted_list}")
    print(f"Počet iterací: {total_iterations}")
