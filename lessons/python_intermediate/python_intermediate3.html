Terminál a barevné animace

Pro vytváření našich projeků budeme používat IDE neboli vývojové prostředí od společnosti Microsoft s názvem Visual Studio Code. Návod na stažení a zprovozněni naleznete zde.

V dnešní lekci si vysvětlíme co je to terminál, jak se používá v rámci použitelnosti s uživatelem. Ukážeme si jak nainstalovat knohovnu Colorama k vylepšení práce v terminálu.

Terminál je speciální nástroj, který nám umožňuje spouštět programy a zobrazovat jejich výsledky. Ve Visual Studio Code (VSC) je terminál přístupný přímo ve spodní části okna a jmenuje se jednoduše TERMINAL.

V terminálu můžeme psát příkazy, které spustí naše programy, nebo dokonce ovládat náš počítač. Například můžeme:

  • Spustit Python skript (např. python moje_programy.py).
  • Zobrazit seznam souborů ve složce.
  • Změnit barvy a funkce terminálu.

Jak otevřít terminál ve VSC?

Je to snadné! Postupuj podle těchto kroků:

  1. Otevři Visual Studio Code.
  2. Klikni na horní menu View (Zobrazit).
  3. Vyber možnost Terminal (Terminál).

Nyní by se ti měl terminál zobrazit ve spodní části okna. Můžeš v něm psát příkazy a vidět výsledky svého programu!

Knihovna os v Pythonu

Knihovna os nám pomáhá pracovat s věcmi, jako jsou soubory, složky a operační systém. Tady je pár užitečných příkazů:

  • os.getcwd(): Vrací, ve které složce právě pracuješ.
    Příklad: Pokud jsi v C:/MojeSoubory, vrátí to právě tuto cestu.
  • os.listdir(): Ukáže seznam všech souborů a složek ve vybrané složce.
    Příklad: Vidíš soubory, jako třeba dokument.txt nebo obrazek.png.
  • os.mkdir(): Vytvoří novou složku.
    Příklad: Pokud napíšeš os.mkdir("NovaSlozka"), vytvoří se složka s názvem NovaSlozka.
  • os.remove(): Smaže soubor.
    Příklad: Když máš soubor test.txt a napíšeš os.remove("test.txt"), soubor se smaže.
  • os.system(): Spouští příkazy operačního systému.
    Příklad: Můžeš spustit příkaz, který změní velikost terminálu nebo něco podobného.
  • Nastavení velikosti terminálu: Změní velikost okna terminálu na určitý počet sloupců a řádků.

    Kód:

    import os

def set_terminal_size(columns, rows):
    if os.name == 'nt':  # Windows
        os.system(f'mode con: cols={columns} lines={rows}')
    else:  # Linux, macOS
        os.system(f'printf "\\e[8;{rows};{columns}t"')

# Nastavení terminálu na 100 sloupců a 30 řádků
set_terminal_size(100, 30)
