Terminál a barevné animace

Pro vytváření našich projektů budeme používat IDE od společnosti Microsoft s názvem Visual Studio Code. Návod na stažení a zprovoznění naleznete zde.

V dnešní lekci si vysvětlíme, co je to terminál a jak se používá v rámci uživatelské interakce. Také ukážeme, jak nainstalovat knihovnu Colorama pro vylepšení práce v terminálu.

Opakování

Tento test je zaměřen na procvičení základních a středně pokročilých konceptů v Pythonu. Ověříte si své dovednosti v oblasti generování náhodných čísel, měření času a manipulace s textovými barvami. Po dokončení testu budete mít možnost zkontrolovat své odpovědi a získat zpětnou vazbu.

Co je to terminál?

Terminál je nástroj, který umožňuje spouštět programy a zobrazovat jejich výsledky. Ve Visual Studio Code (VSC) je terminál přístupný ve spodní části okna a nazývá se jednoduše TERMINAL.

Jak otevřít terminál ve VSC?

Otevření terminálu ve VSC je snadné! Postupuj podle těchto kroků:

  1. Otevři Visual Studio Code.
  2. Klikni na horní menu View (Zobrazit).
  3. Vyber možnost Terminal (Terminál).

Knihovna os v Pythonu

Knihovna os nám pomáhá pracovat s věcmi, jako jsou soubory, složky a operační systém. Zde jsou některé užitečné příkazy:

  • os.getcwd(): Vrací aktuální pracovní adresář.

    Příklad: Pokud je skript ve složce C:/MojeSoubory, program vypíše právě tuto cestu.

    
import os

# Vypíše aktuální pracovní adresář
print(os.getcwd())
  • os.listdir(): Zobrazí seznam souborů v adresáři.

    Příklad: Tento kód vypíše všechny soubory ve složce, ve které se právě nacházíme.

    
import os

# Vypíše všechny soubory ve složce
for file in os.listdir():
print(file)
  • os.mkdir(): Vytvoří novou složku.

    Příklad: Tento kód vytvoří složku s názvem "NovaSlozka" v aktuálním adresáři.

    
import os

# Vytvoří novou složku
os.mkdir('NovaSlozka')
  • os.remove(): Smaže soubor.

    Příklad: Tento kód smaže soubor s názvem "test.txt" v aktuálním adresáři.

    
import os

# Smaže soubor
os.remove('test.txt')
  • os.system(): Spustí příkaz operačního systému.

    Příklad: Tento kód spustí příkaz, který změní velikost terminálu (na Windows).

    
import os

# Změní velikost terminálu
os.system('mode con: cols=100 lines=30')
                
import os

def set_terminal_size(columns, rows):
    if os.name == 'nt':  # Windows
        os.system(f'mode con: cols={columns} lines={rows}')
    else:  # Linux, macOS
        os.system(f'printf "\\e[8;{rows};{columns}t"')

# Nastavení terminálu na 100 sloupců a 30 řádků
set_terminal_size(100, 30)

Instalace a použití knihovny Colorama

Nainstalujte knihovnu Colorama pro barevné texty v terminálu. Následuje příklad použití: 

from colorama import init, Fore, Back, Style

# Inicializace Colorama
init()

# Tisk textu s různými barvami a styly
print(Fore.RED + 'This text is red')
print(Back.GREEN + 'This text has a green background')
print(Style.BRIGHT + 'This text is bright')
print(Style.RESET_ALL + 'Back to normal text')
