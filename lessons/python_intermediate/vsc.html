Prostředí pro vývoj

Je spousta možných vývojových prostředí, které nám umožní vývoj v Pythonu. Ale pro naši lekci vybral jsem právě program od společnosti Microsoft, který se jmenuje Visual Studio Code. Hlavní důvod proč budeme používat tento program je, že nám ulehčí práci a ukáže vám chyby, které jste udělali. Popřípadně vás navede jak dále.

Postup pro spravnou instalaci programu

Níže uvedené body vás navedou ke spravné instalaci vývojového prostředí. Pokud by však něco selhalo co nejrychleji se mi ozvěte. A pokusím vás navést spravným směrem.

Návod na zprovoznění IDE:

  • Nejprve si stáhněte Python pomocí tohoto ODKAZU
  • Stažení vývojového prostředí za pomocí odkazu zde
  • Poté v Rozšíření si stáhněte tyto rozšíření:
    • ms-python.vscode-pylance
    • ms-python.python
  • Po těchto krocích by jste měli mít hotové a nainstalované vývojové prostředí.
