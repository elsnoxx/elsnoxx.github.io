Soubory v pythonu

V této lekci se naučíme, cjak pracujeme se soubory! K vytváření projektů budeme používat šikovné prostředí od Microsoftu s názvem Visual Studio Code. Potřebujete pomoc? Návod na stažení a zprovoznění najdete zde.

Komponenty moderního osobního počítače

Moderní osobní počítače fungují díky důmyslné spolupráci celé skupiny komponent. Tyto se postupně během let technického pokroku zdokonalovaly. Jsou to zejména následující komponenty:

  • Hlavní procesor nebo CPU – je jádrem počítače. Dokáže načítat data a instrukce z disku nebo paměti, následně podle nich pracovat a pak ukládat výsledky do paměti nebo na disk. Mezi základní instrukce, které procesor ovládá, patří logické operace, porovnávání hodnot a matematické operace pro celá i desetinná čísla.
  • RAM paměť nebo operační paměť – velmi rychlá paměť pro uložení instrukcí a dat. Je však závislá na elektrickém napětí a vše zapomene v momentě, kdy počítač vypneme.
  • Pevný disk nebo SSD disk – trvalá paměť, která je sice výrazně pomalejší než RAM paměť, ale uchovává data i bez elektrické energie.
  • Grafická karta nebo GPU – zajišťuje vykreslování textů a obrázků na displeji. U moderních počítačů navíc obvykle obsahuje grafický akcelerátor, který odlehčuje hlavnímu procesoru od náročných 3D výpočtů.
  • Displej – zařízení, které dokáže zobrazovat pixelovou mřížku v nějakém rozlišení. V současnosti jsou běžné displeje s FullHD rozlišením, což je 1920 x 1080 pixelů.
  • Základní deska – představuje jakousi kostru a míchu počítače, přes kterou ostatní komponenty navzájem komunikují.
  • Klávesnice a myš – dvě široce rozšířená zařízení pro ovládání osobního počítače.

Dočasná RAM paměť vs. trvalá disková paměť

Až dosud naše pythonovské programy vůbec nevyužívaly pevný disk pro práci s daty. Samotné programy sice máme uložené na disku jako soubory s příponou .py. Data, se kterými náš program pracuje, pak vznikají až během běhu programu. Jakmile náš program ukončí své provádění, data se zapomenou a již nejsou dostupná. Pokud potřebujeme, aby si nějaký náš program pamatoval data i po ukončení chodu programu a vypnutí počítače, budeme muset pracovat se soubory.

Souborový systém moderních operačních systémů

Moderní operační systémy mají v sobě zabudovaný souborový systém a zároveň obsahují grafické programy pro organizaci souborů a složek, jako je např. File Explorer u Windows, či Finder u MacOS. Tvorci Pythonu si byli plně vědomi, že pro rozmanité užitečné programy bude nutné, aby Python umožňoval práci se soubory ve formě čtení, zapisování a vytváření souborů. Python existuje v verzích pro různé operační systémy i kvůli tomu, aby byla zabezpečena správná podpora pro práci se soubory v návaznosti na detaily jednotlivých operačních systémů.

Práce se soubory v Pythonu

Python poskytuje širokou škálu funkcí pro práci se soubory, které usnadňují manipulaci s daty. Pomocí vestavěného modulu os je možné provádět operace jako navigace v adresářové struktuře, vytváření nebo mazání souborů a složek. Kromě toho modul shutil nabízí funkce pro kopírování, přesouvání a mazání souborů a složek.

Kromě základních operací pro manipulaci s soubory umožňuje Python také práci s binárními a textovými soubory. Funkce jako open() slouží k otevření souboru pro čtení, zápis nebo přidávání dat. Python také nabízí různé způsoby čtení souborů, například read() pro celé soubory nebo readlines() pro čtení po řádcích.

Díky těmto nástrojům je práce s daty v Pythonu jednoduchá a efektivní, což je nezbytné pro vytváření komplexních programů, které se spoléhají na externí soubory pro uložení a načítání informací.

Načítání seznamu výrobců automobilů v Evropě z textového souboru

Nejprve vytvoříme textový soubor s názvem european_car_producers.txt a zapíšeme do něj názvy deseti evropských výrobců automobilů. Pod Windows můžeme použít např. program Notepad a pod macOS TextEdit. Dejte si pozor, aby byl soubor uložen v kódování UTF-8, což je univerzálně spolehlivé kódování, které pokrývá znaky abeced většiny světových jazyků, včetně čínštiny, korejštiny a japonštiny.

Zdrojový kód pro načítání souboru

Nyní uvedeme zdrojový kód programu, který načte náš soubor a vypíše jeho parametry a obsah: 

        
car_file = open("european_car_producers.txt", mode="r")
car_file_content = car_file.read()
print(car_file)
print(car_file_content)
car_file.close()

Soubor otevřeme funkcí open() a zavřeme metodou close(). Je důležité soubor vždy zavřít, aby se nezablokovaly systémové prostředky. Po otevření souboru jeho obsah načítáme metodou read(). Funkci open() voláme se třemi argumenty:

  • První argument je název souboru v uvozovkách.
  • Druhý argument je režim práce se souborem. Zde jsou některé z možností:
Režim Popis
"r" Režim pro čtení (soubor musí existovat).
"w" Režim pro zápis (pokud soubor neexistuje, bude vytvořen; pokud existuje, jeho obsah bude přepsán).
"a" Režim pro přidávání (pokud soubor neexistuje, bude vytvořen; nové údaje budou přidány na konec souboru).
"x" Režim pro exkluzivní zápis (soubor musí být nový, pokud již existuje, dojde k chybě).
"b" Režim pro binární soubory (přidává se k jiným režimům, např. "rb" nebo "wb" pro čtení nebo zápis binárních souborů).
"t" Režim pro textové soubory (tento režim je implicitní, není nutné ho specifikovat).
"r+" Režim pro čtení a zápis (soubor musí existovat).
"w+" Režim pro čtení a zápis (soubor bude vytvořen nebo přepsán).
"a+" Režim pro čtení a přidávání (nové údaje budou přidány na konec souboru).

Převod načteného textu na seznam a výpis v očíslované podobě

Text načtený ze souboru je uložen do textové proměnné. Tento řetězec můžeme převést na seznam pomocí metody split(). Pokud v textovém souboru existují prázdné řádky, budou v Pythonu reprezentovány jako prázdné položky seznamu. 

        
car_file = open("european_car_producers.txt", mode="r")
car_file_content = car_file.read()
car_list = car_file_content.split("\n")
counter = 0
for producer in car_list:
    counter += 1
    print(f"{counter}. {producer}")
car_file.close()

Zapisování do souboru – sekvence náhodných čísel

Vytvoříme nyní program, který do souboru zapíše sekvenci deseti náhodných čísel v rozmezí od 1 do 100. Tento úkol využívá generování náhodných čísel, které jste se naučili v předchozích blokových lekcích. Kód vypadá takto: 

        
import random
my_file = open("random_numbers.txt", mode="w")
for index in range(10):
    my_random_number = random.randint(1, 100)
    my_file.write(str(my_random_number) + "\n")
my_file.close()

Zapisování do souboru – jedinečná sekvence náhodných čísel

V předchozím řešení se náhodná čísla mohou opakovat. Nyní se zaměříme na problém, kde je potřeba generovat jedinečnou sekvenci, kde se žádné číslo neopakuje, například pro vytvoření náhodného pořadí hráčů ve hře nebo pro náhodné pořadí prezentací ve třídě. 

        
import random
my_random_numbers = []
for index in range(10):
    is_member = True
    while is_member:
        my_trial_random_number = random.randint(1, 10)
        if my_trial_random_number not in my_random_numbers:
            my_random_numbers.append(my_trial_random_number)
            is_member = False
print(my_random_numbers)
my_random_numbers_as_text = [str(item) + "\n" for item in my_random_numbers]
my_file = open("random_unique_numbers.txt", mode="w", encoding="utf-8")
my_file.writelines(my_random_numbers_as_text)
my_file.close()
