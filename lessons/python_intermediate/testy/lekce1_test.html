Úkoly v Pythonu

Čas zbývá: 20:00

Úkol 1:

Záměna dvou hodnot

Řešení: Použijte dočasnou proměnnou pro výměnu dvou hodnot.

Použij `temp = a`, `a = b`, `b = temp` pro výměnu hodnot dvou proměnných.

Úkol 2:

Kontrola, zda je číslo sudé nebo liché

Řešení: Pokud je číslo dělitelné 2, je sudé, jinak je liché.

Použij `if cislo % 2 == 0` pro kontrolu sudosti čísla.

Úkol 3:

Výpis prvků seznamu

        fruits = ["jablko", "banán", "pomeranč"]

Řešení: Použijte smyčku k vypsání všech prvků seznamu.

Použij `for item in seznam:` pro iteraci přes seznam a výpis jeho prvků.

Úkol 4:

Načti číslo n od uživatele. Vytvoř trojúhelník hvězdiček s n řádky

Řešení: Použij smyčky k výpisu trojúhelníku hvězdiček.

Použij `for i in range(1, n+1): print('*' * i)` pro výpis trojúhelníku.

Úkol 5:

Načti věk uživatele a podle věku vypiš typ lístku

Řešení: Pokud je věk menší než 18, vypiš "dětský lístek", jinak "dospělý lístek".

Použij `if vek < 18: print('dětský lístek') else: print('dospělý lístek')`.
