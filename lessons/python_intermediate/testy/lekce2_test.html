Úkol 1:

Sečtení všech čísel v seznamu

seznam = [1, 2, 3, 4, 5]

Řešení: Použijte smyčku k sečtení všech prvků v seznamu.

Použij `sum(seznam)` nebo `for num in seznam:` a postupně sčítej hodnoty.

Úkol 2:

Načti jméno uživatele a pozdrav ho

Řešení: Použij funkci input() pro získání jména a vypiš pozdrav.

Použij `jmeno = input('Zadej své jméno: ')` a poté `print(f'Ahoj, {jmeno}!')`.

Úkol 3:

Převod stupňů Celsia na Fahrenheity

Řešení: Použij vzorec F = C * 9/5 + 32.

Použij `f = c * 9/5 + 32` pro převod teploty.

Úkol 4:

Rozhodni, zda číslo je kladné, záporné nebo nula

Řešení: Použij podmínky k rozhodnutí.

Použij `if cislo > 0`, `elif cislo < 0`, `else` pro vyhodnocení.

Úkol 5:

Nakresli obdélník ze znaků hvězdiček o zadané šířce a výšce

Řešení: Použij smyčky k výpisu obdélníku.

Použij `for i in range(vyska): print('*' * sirka)` pro výpis obdélníku.
