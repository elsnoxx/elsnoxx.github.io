Úvod, IDE, Opakování, cyklus

Pro vytváření našich projeků budeme používat IDE neboli vývojové prostředí od společnosti Microsoft s názvem Visual Studio Code. Návod na stažení a zprovozněni naleznete zde.

Konvence pro zápis proměnných

Tato učebnice používá následující konvence pro pojmenování proměnných v kódu:

  • Všechny názvy budou psány v angličtině.
  • Všechny názvy budou psány malými písmeny.
  • Víceslovné názvy budou používat znak podtržítka _ (například first_name).
  • Názvy proměnných budou popisné, aby bylo zřejmé, k čemu proměnná slouží.

Příklady:

  
  # Správné pojmenování
count = 10               # počet čísel
first_name = "John"      # křestní jméno
total_sum = 100          # celkový součet

# Špatné pojmenování
x = 10                   # název není popisný
a = "John"               # název není popisný
sumTotal = 100           # porušuje konvenci malých písmen

Další poznámky:

Používání anglických názvů je důležité, protože to usnadňuje čtení kódu mezinárodním týmům. Použití popisných názvů proměnných také přispívá k lepší čitelnosti kódu, což zvyšuje produktivitu a snižuje pravděpodobnost chyb.

Opakování

Cykly

V programování se občas stane, že potřebujeme udělat stejnou věc několikrát. Například, když chceme seřadit čísla nebo vykonat stejný úkol vícekrát. Abychom se vyhnuli psaní stejného kódu znovu a znovu, používáme cykly. Cykly nám pomáhají opakovat určité příkazy, dokud není splněna nějaká podmínka.

Co je to cyklus?

Cyklus je jako automat, který dělá určité úkoly znovu a znovu, dokud mu neřekneme, aby přestal. Představ si to jako hru, kde se hráč snaží udělat něco, dokud se mu to nedaří.

Cyklus while

Cyklus while je vhodný pro situace, kdy není předem známo, kolikrát bude nutné cyklus opakovat. Tento cyklus se opakuje, dokud je splněna podmínka. Je proto často používán ve spojení s podmíněnými konstrukcemi.

Pomocná proměnná musí být definována a spravována námi, aby bylo možné kontrolovat průběh cyklu. Základní struktura vypadá takto: 

              

                
i = 0
while (podmínka):
    # kód, který se opakuje
    i += 1  # Spravujeme proměnnou

Například, pokud chceme zobrazit číslo i, dokud je menší než 3:

V tomto příkladu cyklus proběhne třikrát, protože i začíná na 0 a pokaždé se zvyšuje o 1. Každé provedení cyklu se nazývá iterace. 

              

                
i = 0
while i < 3:
    print(i)  # Zobrazí číslo
    i += 1    # Zvyšuje i o 1

Cyklus for

Cyklus for je vhodný pro situace, kdy je předem známo, kolikrát bude třeba cyklus opakovat. V Pythonu se iteruje přes kolekce (například seznamy, řetězce) nebo pomocí funkce range().

Pomocná proměnná a její změnu řídí samotný cyklus podle definovaného intervalu. Základní struktura vypadá takto: 

              

                
for prvek in kolekce:
    # kód, který se provede pro každý prvek

Zde je příklad, který ukáže čísla od 0 do 2:

  • Začátek: range(3) vytváří sekvenci čísel od 0 do 2.
  • Podmínka: Cyklus pokračuje, dokud existují prvky k iteraci.
  • Krok: Automaticky postupujeme k dalšímu číslu v sekvenci.
  • Tělo: print(i) zobrazí aktuální číslo.

Cyklus for je velmi efektivní při práci s iterovatelnými objekty a šetří čas při psaní opakujícího se kódu! 

              

                
for i in range(3):  # range(3) generuje čísla 0, 1, 2
    print(i)  # Zobrazí aktuální číslo

Tvorba menu

Začneme s úvodními hlášky našeho programu. Poté si připravíme základními výpisi menu. Menu bude obsahovat nasledujici funkcnosti.

  • [1] Výpis hráčov hokejového klubu
  • [2] Pridanie hráča za člena klubu
  • [3] Vyradenie hráča z členstva v klube
  • [4] Zotriedenie zoznamu hráčov
  • [0] Ukončenie programu

Následně si naprogramujeme proměnnou selection, kterou nastavíme na None. Teď nám ještě zbývá udělůat logika pomocí if, elif a else. Takže pro každou funkčnost meny vypíšeme výber uživatele. 

        

          
print(f"Zvolil si možnosť #{selection}.") # výpis pro výběry 1 až 4
print("Program bol ukončený.") # výpis pro výběr 0
print("Neznáma voľba.") # výpis pro šoppatný výběr

Pokud již máme vše vyzkoušíme si aplikaci zabalit do nekoněcného cyklu. Který bude končít právě pokud uživatel zvolí možnost 0. Předpis cyklu je uveden níže

while selection != 0: # zde bude hlavní logika programu

Vyzkoušej te si co se stane pokud zadáte do programu písmeno. Funguje? Ne nefunguje a proč tomu tak je? Budem muset do naší apliklace ještě přidat jeden if ,který nám bude hlídat jesli vstup je opravdu číslo.

if selection.isnumeric() == True: # výběr funkčnosti else: print("Neznáma voľba.")

Tím pádem jsme udělali hlavní menu funkčí a hlídající špatné vstupy. Teď se teda vrhneme na funkčnost jednotlivých fukcí našeho meny.

  1. Funkčnost 1

    Pro vytvoření funkčnosti budeme potřebovat vytvořít si novou proměnnou s nazvem hockey_club. Ta bude vytvořena na začátku programu. Dále v ifu kde se nachází detekce tohoto vstupu musíme přidat výpis tohoto seznamu.

    print(f"Zvolil jsi možnost #{selection}.") print("Seznam členů klubu je: ") print(hockey_club)
  2. Funkčnost 2

    Pro vytvoření funkčnosti budeme potřebovat vstup od uživatele, který nám zadá název jednoho hráče. Ten poté pomocí funkce append() vložíme do seznamu hockey_club.

    print(f"Zvolil jsi možnost #{selection}.") player_name = input("Zadej jméno hráče... ") hockey_club.append(player_name)
  3. Funkčnost 3

    Pro vytvoření funkčnosti budeme potřebovat vstup od uživatele, který nám zadá pořadové číslo hráče. A poté ho ze seznamu hockey_club odstraníme. Nezapomeňte že programově je potřeba odečíst 1 od vstupu. Také je potřeba před odstraněním zkontrolovat jestli vstup je možný pomocí if.

    print(f"Zvolil jsi možnost #{selection}.") delete_player_number = int(input("Zadej pořadové číslo hráče pro ukončení jeho členství... ")) if 0 < delete_player_number <= len(hockey_club): del hockey_club[delete_player_number - 1] else: print("Chyba: Hráč s tímto číslem neexistuje.")
  4. Funkčnost 4

    Pro vytvoření funkčnosti budeme setřídíme seznam hráčů pomocí funkce Sort(). Poté vypíšeme celý seznam do konzole.

    print(f"Zvolil jsi možnost #{selection}.") hockey_club.sort() print(hockey_club)
