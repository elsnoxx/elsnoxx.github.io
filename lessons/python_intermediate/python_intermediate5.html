Turtle a soubory

Pro vytváření našich úkolů budeme používat IDE neboli vývojové prostředí od společnosti Microsoft s názvem Visual Studio Code. Návod na stažení a zprovozněni naleznete zde.

Opakování

Tento test je zaměřen na procvičení základních a středně pokročilých konceptů v Pythonu. Ověříte si své dovednosti v oblasti generování náhodných čísel, měření času a manipulace s textovými barvami. Po dokončení testu budete mít možnost zkontrolovat své odpovědi a získat zpětnou vazbu.

Rozličné souborové formáty

Operační systém a formáty souborů

  • Operační systém nerozlišuje mezi formáty souborů.
  • Spravuje umístění, atributy (velikost, práva, datum), ale obsah řeší konkrétní aplikace.

Příklady formátů souborů

  • Binární: .DOC (MS Word), .PPTX (PowerPoint XML), .C3P (Construct3).
  • Normované formáty: .HTML, .CSS, .JPG, .PNG.
  • Rozdělení formátů:
    • Textové: Lze číst (např. HTML, CSS).
    • Binární: Nelze přímo interpretovat (např. JPG, MP3).

Bloková schéma práce s počítačem

Uživatel → Aplikace → Operační systém → Hardware.

Správa souborů: Probíhá v rovině operačního systému.

Formát souborů: Řeší aplikace.

Korytnačí grafika a formát textových souborů

Stáhnout soubor

Cíl

Ukládat instrukce pro kreslení do vlastního textového formátu.

Příklad: .turtledraw

Definice formátu

  • F + číslo: Pohyb vpřed.
  • B + číslo: Pohyb vzad.
  • R + číslo: Otočení vpravo o zadaný úhel.
  • L + číslo: Otočení vlevo o zadaný úhel.
  • Oddělovač instrukcí: |.

Ukázka kódu

        
import turtle

# Načtení souboru
drawing_file = open("equilateral_triangle.turtledraw", mode="r", encoding="utf-8")
drawing = drawing_file.read().split("|")
drawing_file.close()

# Nastavení plátna
my_screen = turtle.Screen()
my_screen.setup(600, 600)
my_turtle = turtle.Turtle()

# Zpracování instrukcí
for instruction in drawing:
  if instruction[0] == "F":
      my_turtle.forward(int(instruction[1:]))
  elif instruction[0] == "B":
      my_turtle.backward(int(instruction[1:]))
  elif instruction[0] == "L":
      my_turtle.left(int(instruction[1:]))
  elif instruction[0] == "R":
      my_turtle.right(int(instruction[1:]))

turtle.done()

Cvičení 1

Stáhnout soubor
Načtení a vykreslení obrázku

Vaším úkolem je stáhnout soubor s instrukcemi pro kreslení obrázku ve formátu .turtledraw. Obsah souboru rozdělte na jednotlivé instrukce, které poté použijete k vykreslení obrázku pomocí knihovny turtle.

  1. Stáhněte soubor z přiloženého odkazu.
  2. Načtěte jeho obsah v Pythonu a rozdělte instrukce podle oddělovače |.
  3. Použijte cyklus k postupnému zpracování instrukcí.
  4. Vykreslete celý obrázek pomocí želví grafiky.

Instrukce se řídí těmito pravidly:

  • F + číslo: Pohyb želvy vpřed.
  • B + číslo: Pohyb želvy vzad.
  • R + číslo: Otočení želvy doprava o zadaný úhel.
  • L + číslo: Otočení želvy doleva o zadaný úhel.
  • C + číslo: Kolečko pomocí želvy.
  • PU : Pero nahoru a přestane kreslit.
  • PD : Pero dolu a přestane kreslit.

Programování: Párnost a nepárnost čísel

Logika

  • Číslo je sudé, pokud zbytek po dělení 2 je 0.
  • Číslo je liché, pokud zbytek po dělení 2 není 0.

Ukázka jednoduchého kódu

        
user_number = int(input("Zadej číslo: "))
if user_number % 2 == 0:
    print("Číslo je sudé.")
else:
    print("Číslo je liché.")

Vysvětlení kódu:

V tomto příkladu se nejprve načte číslo zadané uživatelem jako vstup. Pomocí operátoru % (modulo) se zjistí, zda zbytek po dělení 2 je 0. Pokud ano, číslo je sudé, jinak je liché. Výsledek je následně vypsán na obrazovku.

Kresba podle párnosti

            
import turtle

# Nastavení plátna a korytnačky
my_screen = turtle.Screen()
my_screen.setup(600, 600)
my_turtle = turtle.Turtle()
my_turtle.pensize(3)
my_turtle.penup()
my_turtle.goto(-250, -250)
my_turtle.pendown()

# Kresba
for index in range(10):
    my_turtle.forward(500)
    if index % 2 == 0:
        my_turtle.left(90)
        my_turtle.forward(50)
        my_turtle.left(90)
    else:
        my_turtle.right(90)
        my_turtle.forward(50)
        my_turtle.right(90)

turtle.done()

Jak kód funguje:

Tento program kombinuje logiku párnosti a vizualizaci pomocí knihovny turtle. Želva ("turtle") kreslí čáry na základě sudých a lichých indexů v cyklu:

  • Pokud je číslo sudé, želva se otočí doleva (left(90)).
  • Pokud je číslo liché, želva se otočí doprava (right(90)).

Kresba začíná v levém dolním rohu a postupuje směrem nahoru. Každý řádek je posunut o 50 jednotek směrem nahoru a želva se na konci každého řádku otočí podle sudosti nebo lichosti aktuálního indexu cyklu.

Tipy k experimentování:

  • Změňte počet čar, které želva nakreslí, úpravou range(10) na jiný rozsah.
  • Pozměňte délku čar (my_turtle.forward(500)) a sledujte, jak to ovlivní výsledný vzor.
  • Vyzkoušejte jiné úhly otáčení než 90°, například 45° nebo 60°.

Co si odnést:

Tento příklad ukazuje, jak lze pomocí jednoduché logiky a cyklů vytvořit vizuální vzory. Současně si procvičíte práci s knihovnou turtle, což je zábavný způsob, jak se učit programovat.

Načítání náhodných barev ze seznamu a grafika s želvou

Připomeňme si tvar připomínající dlouhého hada z konce šesté lekce. Byl nakreslen jednou barvou. Nyní ho můžeme vylepšit tím, že každý úsek nakreslíme barvou, která bude náhodně vybrána ze seznamu deseti různých barev. V grafice s želvou je možné používat takzvané pojmenované barvy i barvy založené na modelu RGB, což je rozšířený barevný model ve světě počítačů.

Grafika s želvou je založena na knihovně TkInter, která obsahuje mnoho desítek pojmenovaných barev. Jejich přesný přehled si můžete ve vysokém rozlišení prohlédnout na odkazu, který spravuje vysoká škola Wellesley College.

Výstup programu:

Kód programu:

        
import random
import turtle

my_colors = ["gray", "slate blue", "dodger blue", "cyan", "sea green", \
              "gold2", "coral", "bisque2", "ivory4", "DarkOrchid1"]

my_screen = turtle.Screen()
my_screen.setup(600, 600)
my_turtle = turtle.Turtle()
my_turtle.shape("turtle")
my_turtle.pensize(3)
my_turtle.turtlesize(2)
my_turtle.penup()
my_turtle.back(250)
my_turtle.left(90)
my_turtle.back(250)
my_turtle.right(90)
my_turtle.speed(0)
my_turtle.pendown()

for index in range(21):
    my_turtle.pencolor(random.choice(my_colors))
    my_turtle.forward(500)

    if index % 2 == 0:
        my_turtle.left(90)
        my_turtle.pencolor(random.choice(my_colors))
        my_turtle.forward(25)
        my_turtle.left(90)
    else:
        my_turtle.right(90)
        my_turtle.pencolor(random.choice(my_colors))
        my_turtle.forward(25)
        my_turtle.right(90)

turtle.done()

Deset různých barev je uloženo v seznamu s názvem my_colors. Náhodnou barvu vybíráme před každou novou čárou pomocí metody random.choice(), jejímž argumentem je název seznamu.

Opakující se barevné vzory a grafika s želvou

V programu výše jsme barvy vybírali náhodně. Ale co když bychom chtěli kreslit pomocí přesně se opakující sekvence barev? Například by mohlo jít o pojmenované barvy purple1, purple2, purple3, purple4, MediumPurple1, MediumPurple2, MediumPurple3 a MediumPurple4 a tyto bychom chtěli opakovat dookola.

Očekávaný grafický výsledek je zobrazen na další straně. Takového výsledku dosáhneme, pokud programově vytvoříme sekvenci 0, 1, 2, 3, 4, 5, 6, 7, protože to jsou adresní místa v seznamu barev my_colors.

Kód programu:

        
import random
import turtle

my_colors = ["purple1", "purple2", "purple3", "purple4", "MediumPurple1", \
              "MediumPurple2", "MediumPurple3", "MediumPurple4"]

my_screen = turtle.Screen()
my_screen.setup(600, 600)
my_turtle = turtle.Turtle()
my_turtle.shape("turtle")
my_turtle.pensize(3)
my_turtle.turtlesize(2)
my_turtle.penup()
my_turtle.back(250)
my_turtle.left(90)
my_turtle.back(250)
my_turtle.right(90)
my_turtle.speed(0)
my_turtle.pendown()

counter = 0
for index in range(21):
    position = counter % len(my_colors)
    print(position)
    my_turtle.pencolor(my_colors[position])
    counter += 1
    my_turtle.forward(500)

    if index % 2 == 0:
        my_turtle.left(90)
        position = counter % len(my_colors)
        my_turtle.pencolor(my_colors[position])
        counter += 1
        my_turtle.forward(25)
        my_turtle.left(90)
    else:
        my_turtle.right(90)
        position = counter % len(my_colors)
        my_turtle.pencolor(my_colors[position])
        counter += 1
        my_turtle.forward(25)
        my_turtle.right(90)

turtle.done()
