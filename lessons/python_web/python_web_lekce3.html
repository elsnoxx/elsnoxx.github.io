Github Desktop

Github Desktop je aplikace pro správu repozitářů na Githubu. Umožňuje nám jednoduše stahovat, nahrávat a spravovat kód bez znalosti složitých příkazů. Aplikace je dostupná pro Windows a Mac OS. Po stažení a instalaci aplikace se přihlásíme do svého Github účtu a můžeme začít pracovat.

V aplikaci si můžeme zobrazit všechny repozitáře, které máme vytvořené nebo na kterých pracujeme. Můžeme si zobrazit změny, které jsme provedli a nahrát je na server. Aplikace nám také umožní stahovat kód z repozitáře a pracovat s ním offline.

Instalace Github Desktop

Github Desktop je dostupný pro Windows a Mac OS. Pro stažení aplikace navštivte oficiální stránky Github Desktop. Po stažení spusťte instalační soubor a postupujte podle pokynů.

Vytvoření prvního repozitáře

Po instalaci aplikace se přihlaste do svého Github účtu. V aplikaci klikněte na tlačítko "Create new repository". Zadejte název repozitáře, popis a vyberte cestu, kde bude repozitář uložen. Klikněte na tlačítko "Create repository". Repozitář je vytvořen a můžete začít pracovat.

Kdy a na co použít Tuples, Sets, Dictionaries

Tuples – Neměnné a rychlé sekvence

Kdy použít: Když potřebujete neměnné (immutable) sekvence hodnot. Jsou rychlejší než seznamy a lze je použít jako klíče ve slovnících. 


coordinates = (50.07, 14.42)
user_info = ("Alice", 25, "Programmer")

Sets – Unikátní hodnoty bez pořadí

Kdy použít: Když potřebujete unikátní prvky a rychlé vyhledávání. Hodí se pro množinové operace jako sjednocení a průnik. 


unique_numbers = {1, 2, 3, 4, 5}
stopwords = {"a", "the", "and", "is"}
print("the" in stopwords)  # True

Dictionaries – Klíč-hodnota struktura

Kdy použít: Když chcete data párovat klíč-hodnota pro rychlé vyhledávání. Ideální pro ukládání informací a počítání výskytů. 


user = {"name": "Alice", "age": 25, "job": "Programmer"}
word_count = {"apple": 2, "banana": 3, "orange": 1}

Hledání (Search) a třídění (Sort) v Pythonu

Hledání (Search) v seznamech a slovnících

Hledání v Pythonu může být prováděno několika způsoby. Základní metody jsou lineární hledání (O(n)) a binární hledání (O(log n)).

1. Lineární hledání (Linear Search)

Prohledává seznam prvek po prvku. Vhodné pro malé nebo neuspořádané seznamy.

numbers = [3, 7, 2, 9, 4]
    target = 9
    if target in numbers:
        print("Hodnota nalezena!")

2. Binární hledání (Binary Search)

Funguje pouze na seřazených seznamech a je mnohem rychlejší.

import bisect
    numbers = [1, 3, 5, 7, 9]
    index = bisect.bisect_left(numbers, 5)
    print(f"Hodnota 5 je na indexu {index}")

Třídění (Sort) v Pythonu

Python nabízí několik způsobů třídění, od jednoduchého `.sort()` po složitější třídění podle klíčů.

1. Základní třídění pomocí `sort()` a `sorted()`

`sort()` mění původní seznam, zatímco `sorted()` vrací nový seřazený seznam.

numbers = [5, 2, 9, 1, 5]
    numbers.sort()
    print(numbers)  # [1, 2, 5, 5, 9]

2. Třídění podle vlastního klíče

Použití argumentu `key` pro třídění podle specifických kritérií.

people = [("Alice", 25), ("Bob", 30), ("Charlie", 20)]
    people.sort(key=lambda x: x[1])
    print(people)  # Seřazeno podle věku

Shrnutí

Potřebuji... Co použít
Najít prvek v malém nebo neuspořádaném seznamu Lineární hledání (`in`, `.index()`)
Najít prvek v seřazeném seznamu Binární hledání (`bisect` modul)
Seřadit seznam čísel `sort()` nebo `sorted()`
Seřadit seznam objektů podle klíče `sort(key=...)`
