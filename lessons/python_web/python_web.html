Web Development with Python

Vítejte v našem kurzu Web Development with Python!

Cílem tohoto kurzu je naučit vás vytvářet moderní webové aplikace pomocí jazyka Python. Postupně si osvojíte klíčové koncepty, jako je objektově orientované programování, zpracování dat z formulářů, práce s relacemi a principy síťové interakce.

Během kurzu se naučíte používat standardní funkce Pythonu, regulární výrazy, návrhové vzory a systém správy verzí. Dále se seznámíte s frameworky Flask, Bottle a Django, které vám umožní vytvářet robustní a efektivní webové aplikace.

Pro úspěšné absolvování kurzu je nutné odevzdat praktický úkol a složit závěrečný test. K testu budete připuštěni pouze po splnění všech domácích úkolů a praktických úloh.

Před zahájením kurzu doporučujeme projít materiály „Introduction to Python“ z Microsoft Imagine Academy, abyste měli pevné základy pro další studium.

Připravte se na vzrušující cestu plnou objevování a učení! Pokud budete mít jakékoli dotazy nebo budete potřebovat pomoc, neváhejte se na nás obrátit. Přejeme vám hodně úspěchů a radosti z programování!
