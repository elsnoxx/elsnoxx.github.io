Python Web Development: Flask Framework

V této lekci se seznámíme se základy webového frameworku Flask, naučíme se vytvořit jednoduchou webovou aplikaci a pochopíme základní koncepty a příkazy.

Flask je lehký a flexibilní webový framework pro Python, který je ideální pro začátečníky i pokročilé vývojáře.

Co je to Flask?

Flask je mikro-webový framework napsaný v Pythonu. Označení "mikro" neznamená, že celá vaše webová aplikace musí být v jednom souboru, ale spíše to, že Flask se snaží udržet jádro jednoduché, ale přitom rozšiřitelné.

Mezi hlavní výhody Flask patří:

  • Jednoduchý a snadno pochopitelný kód
  • Minimalistický přístup s možností rozšíření
  • Ideální pro malé až středně velké projekty
  • Vynikající dokumentace
  • Aktivní komunita
  • Kompatibilita s mnoha knihovnami a nástroji
Věděli jste? Flask byl vytvořen jako aprílový žert v roce 2010, ale díky své jednoduchosti a eleganci se rychle stal jedním z nejoblíbenějších webových frameworků pro Python.

Instalace a nastavení

Než začneme pracovat s Flaskem, potřebujeme ho nainstalovat. Nejjednodušší způsob je použít správce balíčků pip:

pip install flask
Doporučujeme používat virtuální prostředí pro každý projekt, abyste oddělili závislosti pro různé projekty.

Vytvoření virtuálního prostředí

# Vytvoření virtuálního prostředí
python -m venv venv

# Aktivace virtuálního prostředí (Windows)
venv\Scripts\activate

# Aktivace virtuálního prostředí (macOS/Linux)
source venv/bin/activate

# Nyní nainstalujte Flask
pip install flask

Po instalaci můžeme ověřit, zda je Flask správně nainstalován:

import flask
print(flask.__version__)

Vytvoření první Flask aplikace

Vytvoříme jednoduchý soubor app.py s minimální Flask aplikací:

from flask import Flask

# Vytvoření instance Flask aplikace
app = Flask(__name__)

# Vytvoření routy pro hlavní stránku
@app.route('/')
def hello_world():
    return 'Hello, World!'

# Spuštění aplikace při přímém spuštění tohoto souboru
if __name__ == '__main__':
    app.run(debug=True)

Spuštění aplikace

Aplikaci můžeme spustit přímo z příkazové řádky:

python app.py

Po spuštění aplikace bude dostupná na adrese http://127.0.0.1:5000/ (localhost na portu 5000).

Co se právě stalo?
  1. Vytvořili jsme instanci Flask aplikace
  2. Definovali jsme funkci hello_world() a propojili ji s URL cestou '/'
  3. Spustili jsme vývojový server s aktivovaným režimem ladění
Režim ladění (debug mode)

Parameter debug=True zajistí, že:

  • Server se automaticky restartuje při změnách kódu
  • Zobrazí interaktivní debugger při chybách
  • Poznámka: V produkčním prostředí by měl být vypnutý!

Základní příkazy a koncepty

1. Routy (Routes)

Routy určují, jaká funkce se má spustit při návštěvě konkrétní URL:

@app.route('/about')
def about():
    return 'O nás'

@app.route('/user/')
def show_user(username):
    return f'Uživatel: {username}'

2. Dynamické routy

Flask podporuje dynamické segmenty v URL cestách:

# Cesta s parametrem (int)
@app.route('/post/')
def show_post(post_id):
    return f'Příspěvek číslo: {post_id}'

# Cesta s parametrem (string)
@app.route('/category/')
def show_category(category):
    return f'Kategorie: {category}'

3. HTTP metody

Ve výchozím nastavení reagují routy pouze na GET požadavky. Můžete specifikovat další metody:

@app.route('/login', methods=['GET', 'POST'])
def login():
    if request.method == 'POST':
        # zpracování přihlášení
        return 'Zpracování přihlášení'
    else:
        # zobrazení přihlašovacího formuláře
        return 'Přihlašovací formulář'

4. Templates (Šablony)

Pro vykreslení HTML používá Flask šablonovací systém Jinja2:

from flask import render_template

@app.route('/hello/')
def hello(name):
    return render_template('hello.html', name=name)

Soubor hello.html (v adresáři templates):

<!DOCTYPE html>
<html>
  <head>
    <title>Hello</title>
  </head>
  <body>
    <h1>Hello {{ name }}!</h1>
  </body>
</html>

5. Zpracování formulářů

from flask import request

@app.route('/form', methods=['GET', 'POST'])
def form():
    if request.method == 'POST':
        name = request.form['name']
        return f'Zdravím, {name}!'
    return '''
        <form method="post">
            <input name="name">
            <button type="submit">Odeslat</button>
        </form>
    '''

6. Redirecty a chyby

from flask import redirect, url_for, abort

@app.route('/redirect')
def redirect_example():
    # Přesměrování na jinou stránku
    return redirect(url_for('hello_world'))

@app.route('/error')
def error_example():
    # Vyvolání chyby 404
    abort(404)

# Vlastní stránka pro chybu 404
@app.errorhandler(404)
def page_not_found(e):
    return "Stránka nebyla nalezena", 404
