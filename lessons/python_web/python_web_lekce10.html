✏️ 1. Úvod: Co je databáze a SQL? 👉 Vysvětli: Databáze = místo, kde ukládáme data (tabulky). SQL = jazyk pro práci s daty (dotazování, vkládání, mazání). Tabulka = něco jako tabulka v Excelu, má řádky a sloupce. ✏️ 2. Ukázka základního SELECT Použij Live SQL (https://livesql.oracle.com/ords/f?p=590:41:110367629411516:::41:P41_ID:333667445241506780258350156571847879837) a napiš: sql Copy Edit -- Vypsání všech zaměstnanců SELECT * FROM EMP; -- Vypsání všech oddělení SELECT * FROM DEPT; 3. Vybrané sloupce a WHERE sql Copy Edit -- Vypsat jen jména a platy SELECT ENAME, SAL FROM EMP; -- Vypsat zaměstnance s platem vyšším než 2000 SELECT ENAME, SAL FROM EMP WHERE SAL > 2000; 4. Seskupování a souhrny sql Copy Edit -- Celkový počet zaměstnanců SELECT COUNT(*) FROM EMP; -- Průměrný plat SELECT AVG(SAL) FROM EMP; -- Součet platů SELECT SUM(SAL) FROM EMP; 5. Spojování tabulek (JOIN) sql Copy Edit -- Připojit oddělení k zaměstnancům SELECT E.ENAME, E.SAL, D.DNAME FROM EMP E JOIN DEPT D ON E.DEPTNO = D.DEPTNO; 6. Vkládání dat (INSERT) sql Copy Edit -- Vložit nového zaměstnance INSERT INTO EMP (EMPNO, ENAME, JOB, MGR, HIREDATE, SAL, COMM, DEPTNO) VALUES (9999, 'NOVAK', 'ANALYST', NULL, SYSDATE, 3000, NULL, 10); 7. Smazání a aktualizace (DELETE a UPDATE) sql Copy Edit -- Aktualizace platu zaměstnance UPDATE EMP SET SAL = SAL + 500 WHERE ENAME = 'NOVAK'; -- Smazání zaměstnance DELETE FROM EMP WHERE ENAME = 'NOVAK';

vysvetleni jak tohle budeme pouzivat vramci prace django, jak se vytvari modely, jak se vytvari tabulky, jak se pracuje s daty v ramci djangovych modelu

Vytvoření Django projektu a aplikace V terminálu (nebo VS Code) spusť: bash Copy Edit python -m django startproject mojeaplikace cd moje_aplikace python manage.py startapp zamestnanci 📝 Poznámka: moje_aplikace je název projektu. zamestnanci je název aplikace (část projektu). Nastavení aplikace v settings.py V souboru moje_aplikace/settings.py najdi sekci INSTALLED_APPS a přidej tam svou appku: python Copy Edit INSTALLED_APPS = [ ... 'zamestnanci', ] Vytvoření modelů (models.py) V souboru zamestnanci/models.py napiš: python Copy Edit from django.db import models class Department(models.Model): deptno = models.IntegerField(primary_key=True) dname = models.CharField(max_length=50) def __str__(self): return self.dname class Employee(models.Model): empno = models.IntegerField(primary_key=True) ename = models.CharField(max_length=50) job = models.CharField(max_length=30) mgr = models.IntegerField(null=True, blank=True) hiredate = models.DateField() sal = models.DecimalField(max_digits=10, decimal_places=2) comm = models.DecimalField(max_digits=10, decimal_places=2, null=True, blank=True) deptno = models.ForeignKey(Department, on_delete=models.CASCADE) def __str__(self): return self.ename 📝 Poznámka: __str__ metoda umožní, aby se v adminu a shellu zobrazovalo pěkně jméno (například místo Employee object bude John Smith). ForeignKey propojuje zaměstnance s oddělením. Vytvoření migrací Teď Django přeloží naše modely na SQL příkazy. V terminálu spusť: bash Copy Edit python manage.py makemigrations zamestnanci 👉 Django vypíše něco jako: sql Copy Edit Migrations for 'zamestnanci': zamestnanci/migrations/0001_initial.py - Create model Department - Create model Employee Pak spusť: bash Copy Edit python manage.py migrate 👉 Django provede SQL příkazy v databázi (standardně SQLite) a vytvoří tabulky. řidání modelů do administrace (admin.py) V souboru zamestnanci/admin.py napiš: python Copy Edit from django.contrib import admin from .models import Employee, Department admin.site.register(Employee) admin.site.register(Department) 👉 Tím řekneš Django adminu, aby zpřístupnil tabulky Employee a Department přes webové rozhraní. puštění vývojového serveru V terminálu spusť: bash Copy Edit python manage.py runserver a v prohlížeči otevři: arduino Copy Edit http://127.0.0.1:8000/admin/ Vytvoření administrátorského uživatele Aby ses do adminu mohl přihlásit, musíš vytvořit účet: bash Copy Edit python manage.py createsuperuser 👉 Django se tě zeptá na: Uživatelské jméno (třeba admin) Email (klidně vynech) Heslo Po vytvoření se přihlas do adminu. Naklikání dat V admin rozhraní: ✅ Klikni na Departments → Přidat oddělení (např. 10 – SALES, 20 – IT). ✅ Klikni na Employees → Přidat zaměstnance a vyplň hodnoty (včetně výběru oddělení). Tím dětem ukážeš: Jak tabulky souvisí s modely. Že se data ukládají do databáze a že je můžeš editovat přes klikání (bez psaní SQL).
