Databáze, SQL a Django modely

V této lekci se naučíme, jak pracovat s databázemi pomocí jazyka SQL a jak integrovat databázi do Django aplikace pomocí modelů.

Nejprve si ukážeme základní SQL příkazy pro práci s databází, poté si vytvoříme Django projekt a aplikaci, přidáme modely a nakonec si ukážeme, jak s nimi pracovat v Pythonu.

1. Práce s databází v SQL

  • Databáze = místo, kde ukládáme data (tabulky).
  • SQL = jazyk pro práci s daty (dotazování, vkládání, mazání).
  • Tabulka = něco jako tabulka v Excelu, má řádky a sloupce.
Základní SQL příkazy

Ukázka v online Oracle databázi

-- Vypsání všech zaměstnanců
SELECT * FROM EMP;

-- Vypsání všech oddělení
SELECT * FROM DEPT;
            
-- Vybrané sloupce a WHERE
SELECT ENAME, SAL FROM EMP;
SELECT ENAME, SAL FROM EMP WHERE SAL > 2000;
            
-- Seskupování a souhrny
SELECT COUNT(*) FROM EMP;
SELECT AVG(SAL) FROM EMP;
SELECT SUM(SAL) FROM EMP;
            
-- Spojování tabulek (JOIN)
SELECT E.ENAME, E.SAL, D.DNAME
FROM EMP E
JOIN DEPT D ON E.DEPTNO = D.DEPTNO;
            
-- Vkládání dat (INSERT)
INSERT INTO EMP (EMPNO, ENAME, JOB, MGR, HIREDATE, SAL, COMM, DEPTNO)
VALUES (9999, 'NOVAK', 'ANALYST', NULL, SYSDATE, 3000, NULL, 10);
            
-- Aktualizace a mazání (UPDATE, DELETE)
UPDATE EMP SET SAL = SAL + 500 WHERE ENAME = 'NOVAK';
DELETE FROM EMP WHERE ENAME = 'NOVAK';

2. Nastavení Django projektu a admin menu

  1. Vytvoření projektu a aplikace: 
    django-admin startproject moje_aplikace
cd moje_aplikace
python manage.py startapp zamestnanci
  2. Přidání aplikace do settings.py: 
    INSTALLED_APPS = [
    ...
    'zamestnanci',
]
  3. Modely v models.py: 
    from django.db import models

class Department(models.Model):
    deptno = models.IntegerField(primary_key=True)
    dname = models.CharField(max_length=50)

    def __str__(self):
        return self.dname

class Employee(models.Model):
    empno = models.IntegerField(primary_key=True)
    ename = models.CharField(max_length=50)
    job = models.CharField(max_length=30)
    hiredate = models.DateField()
    sal = models.DecimalField(max_digits=10, decimal_places=2)
    comm = models.DecimalField(max_digits=10, decimal_places=2, null=True, blank=True)
    deptno = models.ForeignKey(Department, on_delete=models.CASCADE)

    def __str__(self):
        return self.ename
  4. Přidání modelů do admin rozhraní (admin.py): 
    # admin.py
from django.contrib import admin
from .models import Employee, Department

admin.site.register(Employee)
admin.site.register(Department)
  5. Migrace a admin menu: 
    python manage.py makemigrations zamestnanci
python manage.py migrate
python manage.py createsuperuser
python manage.py runserver
    Tip: V adminu můžeš přidávat a upravovat data jako v tabulce!

3. Práce s modely a objekty v Pythonu (views)

  • Získání všech zaměstnanců: 
    from .models import Employee

zamestnanci = Employee.objects.all()
for z in zamestnanci:
    print(z.ename, z.sal)
  • Vyhledání podle podmínky: 
    vysoke_platy = Employee.objects.filter(sal__gt=2000)
  • Přidání nového zaměstnance: 
    novy = Employee(
    empno=1001,
    ename="Karel",
    job="Programátor",
    hiredate="2024-06-01",
    sal=3500,
    deptno=Department.objects.get(deptno=10)
)
novy.save()
  • Úprava a mazání: 
    z = Employee.objects.get(empno=1001)
z.sal = 4000
z.save()
z.delete()
Všechny tyto příkazy můžeš použít ve views, shellu nebo v admin rozhraní!
