OOP Python a Opakování HTML

V této lekci se zaměříme na základy objektově orientovaného programování (OOP) v Pythonu a zopakujeme si základy HTML. OOP je důležitý koncept, který nám umožňuje strukturovat kód efektivněji a usnadňuje jeho údržbu.

Základy OOP v Pythonu

Objektově orientované programování (OOP) je způsob, jak strukturovat kód pomocí tříd a objektů. Zde jsou základní koncepty:

1. Třídy a objekty

Třída je šablona pro vytvoření objektů. Objekt je konkrétní instance třídy.


# Definice třídy
class Auto:
    def __init__(self, znacka, barva):
        self.znacka = znacka  # Atribut třídy
        self.barva = barva    # Atribut třídy

    def info(self):  # Metoda třídy
        print(f"Auto: {self.znacka}, Barva: {self.barva}")

# Vytvoření objektu
moje_auto = Auto("Škoda", "modrá")
moje_auto.info()

2. Zapouzdření

Zapouzdření znamená skrytí interních dat objektu. Používáme soukromé atributy (začínající dvěma podtržítky). 


class BankovniUcet:
    def __init__(self, majitel, zustatek):
        self.majitel = majitel
        self.__zustatek = zustatek  # Soukromý atribut

    def vklad(self, castka):
        self.__zustatek += castka

    def get_zustatek(self):
        return self.__zustatek

# Použití
ucet = BankovniUcet("Jan Novák", 1000)
ucet.vklad(500)
print(f"Zůstatek: {ucet.get_zustatek()} Kč")

3. Dědičnost

Dědičnost umožňuje vytvořit novou třídu, která dědí vlastnosti a metody jiné třídy.


class Zvire:
    def __init__(self, jmeno):
        self.jmeno = jmeno

    def zvuk(self):
        pass  # Abstraktní metoda

class Pes(Zvire):
    def zvuk(self):
        return "Haf haf!"

class Kocka(Zvire):
    def zvuk(self):
        return "Mňau!"

# Použití
pes = Pes("Rex")
kocka = Kocka("Micka")
print(f"{pes.jmeno} říká: {pes.zvuk()}")
print(f"{kocka.jmeno} říká: {kocka.zvuk()}")

4. Polymorfismus

Polymorfismus umožňuje, aby stejná metoda fungovala různě podle třídy objektu.


def udelej_zvuk(zvire):
    print(zvire.zvuk())

# Použití
udelej_zvuk(pes)  # Výstup: Haf haf!
udelej_zvuk(kocka)  # Výstup: Mňau!

Základy HTML

HTML (HyperText Markup Language) je základní jazyk pro tvorbu webových stránek. Používá se k vytváření struktury a obsahu webu.

1. Struktura HTML dokumentu

Každý HTML dokument má základní strukturu:


<!DOCTYPE html>
<html lang="cs">
<head>
  <meta charset="UTF-8">
  <title>Název stránky</title>
</head>
<body>
  <h1>Vítejte na mé stránce!</h1>
  <p>Toto je základní HTML dokument.</p>
</body>
</html>

2. Základní HTML tagy

HTML používá značky (tagy) k definování různých částí obsahu. Zde jsou některé základní tagy:

  • <h1> až <h6>: Nadpisy (h1 je největší, h6 nejmenší)
  • <p>: Odstavec
  • <a>: Odkaz
  • <img>: Obrázek
  • <ul> a <li>: Neuspořádaný seznam
  • <ol> a <li>: Uspořádaný seznam
  • <table>: Tabulka

3. Příklad použití základních tagů

Zde je příklad jednoduché webové stránky:


<!DOCTYPE html>
<html lang="cs">
<head>
  <meta charset="UTF-8">
  <title>Moje první stránka</title>
</head>
<body>
  <h1>Vítejte!</h1>
  <p>Toto je moje první webová stránka.</p>
  <a href="https://www.example.com">Klikněte zde pro více informací</a>
  <h2>Obrázek</h2>
  <img src="obrazek.jpg" alt="Popis obrázku" width="300">
  <h2>Seznam</h2>
  <ul>
    <li>Položka 1</li>
    <li>Položka 2</li>
    <li>Položka 3</li>
  </ul>
</body>
</html>

4. Komentáře v HTML

Komentáře se používají k přidání poznámek do kódu. Nejsou viditelné na stránce:


<!-- Toto je komentář -->
<p>Tento text bude viditelný na stránce.</p>

5. Základní atributy

HTML tagy mohou mít atributy, které poskytují další informace:

  • href: Používá se u <a> pro definování odkazu.
  • src: Používá se u <img> pro definování cesty k obrázku.
  • alt: Alternativní text pro obrázek.
  • id: Jedinečný identifikátor prvku.
  • class: Třída prvku pro stylování pomocí CSS.
