ToDo aplikace v Djangu

V této lekci se si vyzkoušíme předělat námi vztvořenou aplikace ve Flasku do Djanga.

Nejprve si ukážeme jak vypadala nase aplikace ve Flasku, kterou jsme si vytvořili v předchozí lekci. A potom se vrhneme na přepis do Djanga.

Úvod - Převod Flask TODO aplikace do Djanga

V předchozí lekci jsme si vytvořili TODO aplikaci ve Flasku. Nyní si ukážeme, jak tutéž aplikaci přepsat do Djanga s využitím databáze místo seznamů v paměti.

Co naše TODO aplikace umí:
  • Přidávání nových úkolů s popisem, poznámkou, očekávaným termínem a předmětem
  • Označování úkolů jako splněných
  • Mazání úkolů
  • Editace existujících úkolů
  • Zobrazení historie splněných a smazaných úkolů
Výhoda Djanga: Data se budou ukládat do databáze, takže se neztratí po restartu serveru!

Krok 1: Inicializace Django projektu

  1. Vytvoření projektu a aplikace: 
    django-admin startproject todo_projekt
cd todo_projekt
python manage.py startapp todo_app
  2. Přidání aplikace do settings.py: 
    INSTALLED_APPS = [
    'django.contrib.admin',
    'django.contrib.auth',
    'django.contrib.contenttypes',
    'django.contrib.sessions',
    'django.contrib.messages',
    'django.contrib.staticfiles',
    'todo_app',  # naše aplikace
]
  3. Vytvoření složek pro šablony a statické soubory: 
    mkdir todo_app/templates
mkdir todo_app/static
mkdir todo_app/static/css
mkdir todo_app/static/js

Krok 2: Vytvoření modelů a jejich migrace

Model pro TODO úkoly:
# todo_app/models.py
from django.db import models
from django.utils import timezone

class Todo(models.Model):
    text = models.CharField(max_length=200, verbose_name="Úkol")
    poznamka = models.TextField(blank=True, verbose_name="Poznámka")
    ocekavane = models.DateField(null=True, blank=True, verbose_name="Očekávaný termín")
    predmet = models.CharField(max_length=100, blank=True, verbose_name="Předmět")
    done = models.BooleanField(default=False, verbose_name="Splněno")
    created = models.DateTimeField(default=timezone.now, verbose_name="Vytvořeno")
    completed = models.DateTimeField(null=True, blank=True, verbose_name="Dokončeno")
    deleted = models.BooleanField(default=False, verbose_name="Smazáno")
    deleted_at = models.DateTimeField(null=True, blank=True, verbose_name="Smazáno v")
    
    class Meta:
        verbose_name = "TODO úkol"
        verbose_name_plural = "TODO úkoly"
        ordering = ['-created']
    
    def __str__(self):
        return self.text
Spuštění migrací:
python manage.py makemigrations todo_app
python manage.py migrate
Vysvětlení polí:
- text: Hlavní popis úkolu
- poznamka: Volitelná poznámka
- ocekavane: Termín splnění
- predmet: Kategorie/předmět úkolu
- done: Zda je úkol splněn
- deleted: Soft delete - úkol není skutečně smazán

Krok 3: Vytvoření superuživatele a admin rozhraní

  1. Vytvoření superuživatele: 
    python manage.py createsuperuser
    Django se zeptá na uživatelské jméno, email a heslo. Heslo musí být dostatečně silné!
  2. Registrace modelu v admin rozhraní: 
    # todo_app/admin.py
from django.contrib import admin
from .models import Todo

@admin.register(Todo)
class TodoAdmin(admin.ModelAdmin):
    list_display = ['text', 'predmet', 'ocekavane', 'done', 'created']
    list_filter = ['done', 'deleted', 'predmet', 'created']
    search_fields = ['text', 'poznamka']
    list_editable = ['done']
    date_hierarchy = 'created'
  3. Spuštění serveru a přístup k adminu: 
    python manage.py runserver

    Přejdi na http://127.0.0.1:8000/admin/ a přihlas se svými údaji.

Krok 4: Popis endpointů a jejich funkce

Přehled všech URL adres, které budeme vytvářet:
URL Název Metody Funkce
/ home GET, POST Hlavní stránka - zobrazení úkolů a přidání nového
/done/<int:pk>/ done POST Označení úkolu jako splněného
/delete/<int:pk>/ delete GET Smazání (soft delete) úkolu
/edit/<int:pk>/ edit GET, POST Editace existujícího úkolu
/history/ history GET Zobrazení splněných a smazaných úkolů
Ukázka URL konfigurace:
# todo_app/urls.py
from django.urls import path
from . import views

app_name = 'todo_app'
urlpatterns = [
    path('', views.index, name='home'),
    path('done/<int:pk>/', views.done, name='done'),
    path('delete/<int:pk>/', views.delete, name='delete'),
    path('edit/<int:pk>/', views.edit, name='edit'),
    path('history/', views.history, name='history'),
]
# todo_projekt/urls.py
from django.contrib import admin
from django.urls import path, include

urlpatterns = [
    path('admin/', admin.site.urls),
    path('', include('todo_app.urls')),
]
