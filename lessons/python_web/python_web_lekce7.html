Python Web Development: Flask Framework

V této lekci se seznámíme se základy webového frameworku Flask, naučíme se vytvořit jednoduchou webovou aplikaci a pochopíme základní koncepty a příkazy.

Flask je lehký a flexibilní webový framework pro Python, který je ideální pro začátečníky i pokročilé vývojáře.

Co je to Flask?

Flask je mikro-webový framework napsaný v Pythonu. Označení "mikro" neznamená, že celá vaše webová aplikace musí být v jednom souboru, ale spíše to, že Flask se snaží udržet jádro jednoduché, ale přitom rozšiřitelné.

Mezi hlavní výhody Flask patří:

  • Jednoduchý a snadno pochopitelný kód
  • Minimalistický přístup s možností rozšíření
  • Ideální pro malé až středně velké projekty
  • Vynikající dokumentace
  • Aktivní komunita
  • Kompatibilita s mnoha knihovnami a nástroji
Věděli jste? Flask byl vytvořen jako aprílový žert v roce 2010, ale díky své jednoduchosti a eleganci se rychle stal jedním z nejoblíbenějších webových frameworků pro Python.

Práce s HTML šablonami ve Flasku (Jinja2)

Flask používá šablonovací systém Jinja2 pro generování dynamických HTML stránek. Šablony jsou běžně ukládány do složky templates v kořenovém adresáři projektu.

Struktura projektu:
my_flask_app/
├── app.py
└── templates/
    └── index.html
Příklad použití render_template:
from flask import Flask, render_template

app = Flask(__name__)

@app.route('/')
def home():
    return render_template('index.html', name='Svět')

if __name__ == '__main__':
    app.run(debug=True)
Obsah souboru templates/index.html:
<!DOCTYPE html>
<html lang="cs">
  <head>
    <meta charset="UTF-8">
    <title>Úvodní stránka</title>
  </head>
  <body>
    <h1>Vítej, {{ name }}!</h1>
    <p>Toto je stránka vykreslená pomocí Flasku a Jinja2 šablony.</p>
  </body>
</html>
Tip: Pomocí složky templates můžete vytvářet více HTML souborů a dynamicky je vykreslovat podle potřeby. V šablonách lze používat proměnné, cykly a podmínky.
Další možnosti Jinja2:
  • Používání proměnných: {{ promenna }}
  • Podmínky: {% if ... %} ... {% endif %}
  • Cyklus: {% for item in seznam %} ... {% endfor %}
  • Vkládání dalších šablon: {% include 'footer.html' %}
  • Dědičnost šablon: {% extends 'base.html' %}

Dynamický web a generování obsahu ve Flasku

Moderní webové aplikace nejsou jen statické stránky, ale často generují svůj obsah dynamicky podle požadavků uživatele, dat z databáze nebo jiných vstupů. Flask umožňuje jednoduše vytvářet dynamické weby pomocí tzv. šablon a předávání proměnných z Pythonu do HTML.

Jak funguje generování dynamického webu ve Flasku?
  • Uživatel odešle požadavek na určitou stránku (např. /uzivatele).
  • Flask v Pythonu zpracuje tento požadavek, připraví data (např. seznam uživatelů z databáze).
  • Pomocí funkce render_template předá data do HTML šablony.
  • HTML šablona (Jinja2) vygeneruje výslednou stránku, která se zobrazí uživateli.
Příklad – seznam uživatelů:
from flask import Flask, render_template

app = Flask(__name__)

@app.route('/uzivatele')
def uzivatele():
    users = ["Anna", "Petr", "Jana"]
    return render_template('uzivatele.html', users=users)

Soubor templates/uzivatele.html:

<h2>Seznam uživatelů</h2>
<ul>
  {% for user in users %}
    <li>{{ user }}</li>
  {% endfor %}
</ul>
Tip: Pomocí šablon můžete jednoduše generovat tabulky, seznamy, formuláře nebo celé stránky podle dat, která připravíte v Pythonu.
Co dalšího lze dynamicky generovat?
  • Obsah podle přihlášeného uživatele
  • Výsledky vyhledávání
  • Tabulky z databáze
  • Grafy, statistiky, reporty
  • Stránky s formuláři a jejich zpracování

Díky dynamickému generování obsahu můžete vytvářet interaktivní a užitečné webové aplikace, které reagují na potřeby uživatelů a pracují s aktuálními daty.

Další ukázky generování obsahu v HTML šabloně (Jinja2)

1. Podmíněné zobrazení obsahu

Pomocí if lze zobrazit různé části stránky podle dat z Pythonu:

{% if users %}
  <ul>
    {% for user in users %}
      <li>{{ user }}</li>
    {% endfor %}
  </ul>
{% else %}
  <p>Žádní uživatelé nebyli nalezeni.</p>
{% endif %}
2. Generování tabulky z dat

Tabulku lze jednoduše vytvořit cyklem přes seznam slovníků:

<table class="table">
  <thead>
    <tr><th>Jméno</th><th>Věk</th></tr>
  </thead>
  <tbody>
    {% for osoba in osoby %}
      <tr>
        <td>{{ osoba.jmeno }}</td>
        <td>{{ osoba.vek }}</td>
      </tr>
    {% endfor %}
  </tbody>
</table>

Python kód:

osoby = [
    {"jmeno": "Anna", "vek": 23},
    {"jmeno": "Petr", "vek": 31}
]
return render_template("tabulka.html", osoby=osoby)
3. Vkládání jiné šablony (include)

Části stránky lze rozdělit do více souborů a vkládat je podle potřeby:

{% include 'footer.html' %}
4. Dědičnost šablon (extends)

Pro větší projekty je vhodné mít základní šablonu a z ní dědit:

{% extends 'base.html' %}
{% block content %}
  <h1>Toto je obsah stránky</h1>
{% endblock %}

V base.html musí být definován blok:

<body>
  {% block content %}{% endblock %}
</body>
Tip: Pomocí těchto technik můžete vytvářet přehledné, znovupoužitelné a dynamické šablony pro celý web.
