Lekce 1 - Python

Pro vytváření našich projeků budeme používat IDE neboli vývojové prostředí od společnosti JET BRAINS s nazvem PyCharm. Návod na staženi a zprovozněni naleznete zde.

Proč python?

Vysvětlit proč právě Python by tady bylo na velmi dlouho, ale tady jsou pár důvodu proč se ho budeme učit.

  • jednoduchý pro naučení
  • jednoduchý syntax
  • Je mozne pomocí něj vytvořit spousty veci, jako je web, aplikace pro PC i mobil
  • Začátek s Pythone je velmi jednoduchý

Prostředí pro vývoj

Je spousta možných vývojových prostředí, které nám umožní vývoj v Pythonu. Ale pro naši lekci vybral jsem právě program od společnosti JET BRAINS, který se jmenuje PyCharm. Hlavní důvod proč budeme používat tento program je, že nám ulehčí práci a ukáže vám chyby, které jste udělali. Popřípadně vás navede jak dále.
