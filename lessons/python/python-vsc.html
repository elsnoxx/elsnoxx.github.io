Prostředí pro vývoj

Je spousta možných vývojových prostředí, které nám umožní vývoj v Pythonu. Ale pro naši lekci vybral jsem právě program od společnosti JET BRAINS, který se jmenuje PyCharm. Hlavní důvod proč budeme používat tento program je, že nám ulehčí práci a ukáže vám chyby, které jste udělali. Popřípadně vás navede jak dále.

Postup pro spravnou instalaci programu

Níže uvedené body vás navedou ke spravné instalaci vývojového prostředí. Pokud by však něco selhalo co nejrychleji se mi ozvěte. A pokusím vás navést spravným směrem.

  • Před stažením pyCharmu, prosím zkonttolujte zdali se nachazí Python na vašem PC. Pokud ne postupujte podel těchto kroků:
    • Nejprve si stáhněte Python pomocí tohoto ODKAZU
    • Při instalaci prosím zatrhněte volbu add to enviroment variables, ukazka je nize
  • Stažení vývojového prostředí za pomocí odkazu zde, a
  • Spustit instalační program, který jste stáhli
    pokud na vás vyskočí tohle okno navolte to tak jak je na obrazku
  • Po těchto krocích by jste měli mít hotové a nainstalované vývojové prostředí.
