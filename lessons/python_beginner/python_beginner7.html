Vytvoření hry Wizard Duel

Vytvořme hru nazvanou Wizard Duel a nastavme minimální a maximální hodnoty zdraví, stejně jako vytvoření kouzel pro útok a rušení útoků.

Nejprve si rozluštíme kód uvedený v Seznamu 1 (na konci této lekce), který implementuje duel kouzelníků pomocí jazyka Python.

Seznamy. Vnořené Seznamy

Vytvořme vnořený seznam. Jedná se o novou strukturu, se kterou se seznámíme. Stručně řečeno, vnořený seznam je speciální seznam obsahující další seznamy. K přístupu k prvku v seznamu budete potřebovat ne jedno, ale dva indexy. První je index vnořeného seznamu, druhý je prvek vnořeného seznamu:

numbers = [[1, 2, 3], [10, 20, 30], [100, 200, 300]]
numbers[0] # numbers[0] = [1, 2, 3]
numbers[0][0] # numbers[0][0] = 1

Seznam kouzel, které používají kouzelníci, má následující vlastnosti: název, množství poškození a léčení. Použijeme kouzla uvedená v tabulce při psaní programu. Každé kouzlo má určitou vlastnost, například způsobuje poškození, léčí nebo ruší útoky.

Implementace

Každé kouzlo je prezentováno ve formě seznamu a všechny jsou kombinovány do vnořeného seznamu `spells`:

spells = [['fireball', 10, 0], ['metabolism', 0, 8], ['silence', 0, 0]]

Rozdělme vlastnosti kouzel: první prvek je název, druhý je poškození, třetí je léčení. Prosím, vezměte na vědomí, že existuje i jiný způsob, jak přistupovat k prvkům vnořeného seznamu:

Střílení kouzel

Podmínka hry má velmi důležitý bod: ticho ruší jakýkoli útok a přesměrovává poškození na nepřítele. To se však netýká léčení. Pokud jak počítač, tak hráč vyberou ticho, další provádění kódu nemá smysl, proto je rozumné použít operátor `continue`:

if play_1 == 'silence' and play_2 == 'silence':
    continue

Dále zkontrolujte podmínku. Pokud jeden z kouzelníků rozhodne rušit útok, program musí toto rozpoznat a neodebrat body zdraví hráči:

elif play_1 == 'silence':
    player_select = bot_select
    player_health += spells[bot_select][damage]
elif play_2 == 'silence':
    bot_select = player_select
    bot_health += spells[player_select][damage]

Programování maximálních a minimálních hodnot zdraví

Ujistěte se, že hodnota zdraví nepřesáhne maximální nebo nebude pod 0. Takže vytvořte dodatečnou kontrolu:

if player_health > max_health and bot_health > max_health:
    player_health = max_health
    bot_health = max_health
elif player_health > max_health:
    player_health = max_health
elif bot_health > max_health:
    bot_health = max_health

Pokud je zdraví kouzelníka nižší než 0, hra je u konce a vypíše informace o vítězi:

if player_health < min_health or bot_health < min_health:
    break
