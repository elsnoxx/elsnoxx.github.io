Lekce 5 - Python

Pro vytváření našich projeků budeme používat IDE neboli vývojové prostředí od společnosti Microsoft s názvem Visual Studio Code. Návod na stažení a zprovozněni naleznete zde.

Seznam

V Pythonu jsou seznamy jednou z nejdůležitějších datových struktur. Seznam je kolekce prvků, které mohou být různých typů a jsou uloženy v určeném pořadí. Seznamy jsou dynamické, což znamená, že mohou mít různé délky, a mohou být měněny (mutable), což znamená, že můžete přidávat, odebírat nebo měnit jejich prvky.

V Pythonu můžete seznamy vytvořit pomocí hranatých závorek [ ] a oddělit jednotlivé prvky čárkami. Například: 

          
Muj_seznam = [1, 2, 3, 4, 5]

Vypsání seznamu

Seznam jde vypsat několika způsoby níže je ukázka pár takových vúpisů:

  • Celý list na jednou 
                  
Muj_seznam = [1, 2, 3, 4, 5]
print(Muj_seznam)
  • Jednotlive prvky seznamu 
                  
Muj_seznam = [1, 2, 3, 4, 5]
print(Muj_seznam[0])
  • Pomocí cyklu všechny prvky seznamu 
                  
Muj_seznam = [1, 2, 3, 4, 5]
for prvek in Muj_seznam:
print(prvek)

Operace se seznamy

V nyní se zaměříme na některé běžné operace prováděné se seznamy.

Změna prvků

Můžete přímo přiřadit novou hodnotu na daný index v seznamu. 

            
my_list = [1, 2, 3, 4, 5]
my_list[2] = 10
print(my_list)

Výsledek: [1, 2, 10, 4, 5]

Přidání prvků do seznamu

Můžete přidávat prvky na konec seznamu pomocí metody append(). 

            
Muj_seznam = [1, 2, 3]
Muj_seznam.append(4)
print(Muj_seznam)

Výsledek: [1, 2, 3, 4]

Spojení dvou seznamů

Můžete spojit dva seznamy za pomocí metody extend(). 

            
Muj_seznam = [1, 2, 3]
Druhy_seznam = [4, 5]
Muj_seznam.extend(Druhy_seznam)
print(Muj_seznam)

Výsledek: [1, 2, 3, 4, 5]

Odebírání prvků na pozici/indexu

Můžete odstranit prvky podle hodnoty pomocí metody pop(). Ta nám odstraní číslo, které se nachází na zadané pozici. 

                          
popped_element = my_list.pop(1)
print(my_list)
print("Odebraný prvek:", popped_element)

Výsledek: [1, 3, 4, 5]

Odebírání prvků s zadanou hodnotou

Můžete odstranit prvky podle hodnoty pomocí metody remove(). Ta nám odstraní číslo, které se stejnou hodnotou. 

                          
my_list = [1, 2, 3, 4, 5]
my_list.remove(3)
print(my_list)

Výsledek: [1, 2, 4, 5]

Slicing - rozřezání

Slicing umožňuje vytvářet nové seznamy z již existujícího seznamu. 

            
my_list = [1, 2, 3, 4, 5]
novy_list = my_list[1:4]
print(novy_list)

Výsledek: [2, 3, 4]

Řazení prvků v seznamu

Můžete seřadit prvky seznamu pomocí metody sort(). Tato metoda přeuspořádá prvky seznamu tak, aby byly ve vzestupném pořadí. 

                              
my_list = [5, 2, 8, 1, 4]
my_list.sort()
print(my_list)

Výsledek: [1, 2, 4, 5, 8] 
                              
letters = ["c", "a", "d", "b"]
letters.sort()
print(letters)

Výsledek: ['a', 'b', 'c', 'd']

Generování hodnot do seznamu

Teď si ukážeme jak vytvořit prázdný seznam a následně naplnit ho random hodnotami. 

                        
my_list = [5, 2, 8, 1, 4]
my_list.sort()
print(my_list)

Výsledek: [1, 2, 4, 5, 8]

Domácí úkol

Vašim úkolem bude podle níže uvedených kroků, prokázat znalost dnešní látky.

Postup:
  1. Vytvoří prázdný seznam.
  2. Přidání 20 random prvků do tohoto seznamu.
  3. Změní hodnotu na indexu prvku podle: funkce zde
    • na indexu 1 bude 100
    • na indexu 5 bude 200
    • na indexu 10 bude 300
    • na indexu 15 bude 400
    vypište seznam
  4. Odstraní prvek s hodnotou podle:
    • 100
    • 200
    • 300
    • 400
    vypište seznam
  5. Vytvoří nový seznam pomocí slicingu, který bude v rozmezí 5 až 15 a vypište seznam.
  6. Vytvořte nový seznam, který obsahuje 10 písmen, a seřaď te je abecedně.
      Pro generovaní náhodných písmen použijte kód níže. Ten vám naplní list random písmeny. 
                           
import random
import string

text = []
for prvek in range(10):
    nahodne_pismeno = random.choice(string.ascii_lowercase)
    text.append(nahodne_pismeno)

  • Vygenerované random písmena - ['k', 'i', 'y', 'p', 'w', 'e', 't', 'd', 'o', 'r']
