Lekce 3 - Podmínky a nekonečný cyklus

Pro vytváření našich projeků budeme používat IDE neboli vývojové prostředí od společnosti JET BRAINS s nazvem PyCharm. Návod na staženi a zprovozněni naleznete zde.

V dnešní lekci si ukážeme jednu fantastickou věc, kterou má Python zabudovanou. A to je krestlení za pomocí želvy nebo jiného tvaru. Také se podíváma a vysvětlíme si jak fungují cykly.

Podmínky - konstrukce

Podmínky si vysvětlíme na příkladu.

Máme nějakou osobu. Té se zeptáme jestli je bojovník. Teď od osoby můžeme očekávat 2 odpovědi. A to Ano a Ne. Pokud nám vyjde Ano tak osobe dáme mec. Pokud osoba odpoví Ne pošleme ji domů.

if osoba == válečník:
  zbraň = meč
else:
  print("utíkej domů a schovej se")

Logické operátory

Níže je senzam logických operátorů, které se používají u logických konstrukcí. Fungují Stejně jako v matematice a vrací nám hodnotu True nebo False (pravde a nepravda). Pomocí těchto operátorů jsme schopni porovnávat a řídit logicky kam se kód má ubíhat.

Operátor Popis Příklad Výsledek
> Větší jako print(5>2) True
< Menší jako print(5<2) false
>= Větší nebo rovno print(5>=2) True
<= Menší nebo rovno print(5<=2) false
== Rovný print(5==2) false
print(5==ř) True
!= není rovný print(5!=2) True

Nekonečný cyklus

V minulé lekci jsme si ukázali jak na cykly. V dnešní lekci si ukážeme, jak uděláme nekonečný cyklus a za pomocí podmínky mu dáme možnost skončit. Toho docílíme pomocí break funkce.

Konstrukce

Níže se nachází konstrukce nekonečného cyklu.

while True:
  tělo cyklu

Funkci Break si ukážeme na příkladě počítání do 100.

pocet = 0
while True:
  pocet += 1
  print(pocet)
  if pocet == 100:
    break
