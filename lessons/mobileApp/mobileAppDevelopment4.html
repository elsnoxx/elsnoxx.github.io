Úvod, IDE, Opakování, cyklus

V této lekci si zopakujeme základy programování v Pythonu a naučíme se, jak vytvořit základní menu. Předvedeme si také cykly while a for. A nastavím si konveci pro psaní kódu. Pro vytváření našich projeků budeme používat IDE neboli vývojové prostředí od společnosti Microsoft s názvem Visual Studio Code. Návod na stažení a zprovozněni naleznete zde.

Opakování

Tento test je zaměřen na procvičení základních a středně pokročilých konceptů v Pythonu. Ověříte si své dovednosti v oblasti generování náhodných čísel, měření času a manipulace s textovými barvami. Po dokončení testu budete mít možnost zkontrolovat své odpovědi a získat zpětnou vazbu.

Konvence pro zápis proměnných

Tato učebnice používá následující konvence pro pojmenování proměnných v kódu:

  • Všechny názvy budou psány v angličtině.
  • Všechny názvy budou psány malými písmeny.
  • Víceslovné názvy budou používat znak podtržítka _ (například first_name).
  • Názvy proměnných budou popisné, aby bylo zřejmé, k čemu proměnná slouží.

Příklady:

        
  # Správné pojmenování
count = 10               # počet čísel
first_name = "John"      # křestní jméno
total_sum = 100          # celkový součet

# Špatné pojmenování
x = 10                   # název není popisný
a = "John"               # název není popisný
sumTotal = 100           # porušuje konvenci malých písmen

Další poznámky:

Používání anglických názvů je důležité, protože to usnadňuje čtení kódu mezinárodním týmům. Použití popisných názvů proměnných také přispívá k lepší čitelnosti kódu, což zvyšuje produktivitu a snižuje pravděpodobnost chyb.

Opakování

Cykly

V programování se občas stane, že potřebujeme udělat stejnou věc několikrát. Například, když chceme seřadit čísla nebo vykonat stejný úkol vícekrát. Abychom se vyhnuli psaní stejného kódu znovu a znovu, používáme cykly. Cykly nám pomáhají opakovat určité příkazy, dokud není splněna nějaká podmínka.

Co je to cyklus?

Cyklus je jako automat, který dělá určité úkoly znovu a znovu, dokud mu neřekneme, aby přestal. Představ si to jako hru, kde se hráč snaží udělat něco, dokud se mu to nedaří.

Cyklus while

Cyklus while je vhodný pro situace, kdy není předem známo, kolikrát bude nutné cyklus opakovat. Tento cyklus se opakuje, dokud je splněna podmínka. Je proto často používán ve spojení s podmíněnými konstrukcemi.

Pomocná proměnná musí být definována a spravována námi, aby bylo možné kontrolovat průběh cyklu. Základní struktura vypadá takto: 

            
i = 0
while (podmínka):
    # kód, který se opakuje
    i += 1

Například, pokud chceme zobrazit číslo i, dokud je menší než 3:

V tomto příkladu cyklus proběhne třikrát, protože i začíná na 0 a pokaždé se zvyšuje o 1. Každé provedení cyklu se nazývá iterace. 

              
i = 0
while i < 3:
    print(i)  # Zobrazí číslo
    i += 1    # Zvyšuje i o 1

Cyklus for

Cyklus for je vhodný pro situace, kdy je předem známo, kolikrát bude třeba cyklus opakovat. V Pythonu se iteruje přes kolekce (například seznamy, řetězce) nebo pomocí funkce range().

Pomocná proměnná a její změnu řídí samotný cyklus podle definovaného intervalu. Základní struktura vypadá takto: 

            
for prvek in kolekce:
    # cyklus pro všechny prvky

Zde je příklad, který ukáže čísla od 0 do 2:

  • Začátek: range(3) vytváří sekvenci čísel od 0 do 2.
  • Podmínka: Cyklus pokračuje, dokud existují prvky k iteraci.
  • Krok: Automaticky postupujeme k dalšímu číslu v sekvenci.
  • Tělo: print(i) zobrazí aktuální číslo.

Cyklus for je velmi efektivní při práci s iterovatelnými objekty a šetří čas při psaní opakujícího se kódu! 

            
# range(3) generuje čísla 0, 1, 2
for i in range(3): 
    # Zobrazí aktuální číslo 
    print(i)

Cvičení 2

Průměr z X čísel

Napiš jednoduchý program, který se uživatele zeptá na číslo kolik vstupů bude, a pak všechna čísla sečte vydělí počtem vstupů a vypíše sumu a průměr. Můžeš použít cyklus for nebo while k tomu, aby se program ptal znovu a znovu. Nakonec program spočítá součet a zobrazí ho.

Tvorba menu

Začněme tím, že si vytvoříme úvodní hlášky našeho programu. Poté si naprogramujeme základní menu, které bude obsahovat následující funkčnosti:

  • [1] Výpis hráčů hokejového klubu
  • [2] Přidání hráče do klubu
  • [3] Vyřazení hráče z klubu
  • [4] Seřazení seznamu hráčů
  • [0] Ukončení programu

Pro ovládání menu vytvoříme proměnnou selection, kterou nastavíme na None. Tato proměnná bude obsahovat výběr uživatele. Následně použijeme podmínky if, elif a else k implementaci jednotlivých možností menu. 

        
print(f"Zvolil jsi možnost #{selection}.")  # Výpis pro výběry 1 až 4
print("Program byl ukončen.")               # Výpis pro výběr 0
print("Neznámá volba.")                     # Výpis pro neplatný výběr

Program následně zabalíme do nekonečné smyčky, která poběží, dokud uživatel nezvolí možnost 0 (ukončení programu). 

        
while selection != 0:
    # Zde bude hlavní logika programu

Vyzkoušejte, co se stane, pokud zadáte do programu písmeno místo čísla. Program pravděpodobně skončí chybou. Abychom takovým situacím předešli, přidáme podmínku, která zkontroluje, zda je vstup platné číslo. 

        
if selection.isnumeric():
    # Výběr funkčnosti
else:
    print("Neznámá volba.")

Tímto způsobem jsme vytvořili základní menu, které zvládá kontrolu nesprávných vstupů. Nyní se pustíme do implementace jednotlivých funkcí menu.

  1. Funkce 1: Výpis hráčů

    Nejprve vytvoříme seznam hockey_club, který bude obsahovat hráče klubu. Tuto proměnnou inicializujeme na začátku programu. V rámci výběru 1 vypíšeme aktuální seznam hráčů. 

                
print(f"Zvolil jsi možnost #{selection}.")
print("Seznam členů klubu je:")
print(hockey_club)
  2. Funkce 2: Přidání hráče

    Pro přidání hráče požádáme uživatele o vstup (jméno hráče), který poté pomocí funkce append() přidáme do seznamu hockey_club. 

                
print(f"Zvolil jsi možnost #{selection}.")
player_name = input("Zadej jméno hráče: ")
hockey_club.append(player_name)
  3. Funkce 3: Vyřazení hráče

    Pro vyřazení hráče si od uživatele vyžádáme pořadové číslo hráče v seznamu. Hráče poté odstraníme pomocí del, přičemž před odstraněním zkontrolujeme, zda je zadané číslo platné. Nezapomeňte, že indexy v seznamu začínají od 0, proto musíme od čísla odečíst 1. 

                
print(f"Zvolil jsi možnost #{selection}.")
delete_player_number = int(input("Zadej pořadové číslo hráče: "))

if 0 < delete_player_number <= len(hockey_club):
    del hockey_club[delete_player_number - 1]
else:
    print("Chyba: Hráč s tímto číslem neexistuje.")
  4. Funkce 4: Seřazení hráčů

    Pro seřazení seznamu hráčů použijeme metodu sort(). Tím hráče seřadíme podle abecedy. Následně vypíšeme aktuální seznam. 

                
print(f"Zvolil jsi možnost #{selection}.")
hockey_club.sort()
print("Seznam hráčů byl seřazen:")
print(hockey_club)

Po splnění těchto kroků máte plně funkční aplikaci s ovládacím menu. Vyzkoušejte si přidat, vypsat, odstranit nebo seřadit hráče a ověřte, že vše funguje, jak má.
