Seznamy, Měření času, Buldozer a funkce Block

V této lekci se naučí pracovat se seznamy v Pythonu a měřit čas pomocí knihovny time. Také se dozvíte, jak vytvořit program, který vykresluje bloky na základě uživatelského vstupu a běží v nekonečném cyklu, dokud není ukončen specifickým vstupem. Také se dozvíte jak vytvořit program, který bude fungovat jako buldozer.

Seznamy

V Pythonu jsou seznamy jednou z nejdůležitějších datových struktur. Seznam je kolekce prvků, které mohou být různých typů a jsou uloženy v určeném pořadí. Seznamy jsou dynamické, což znamená, že mohou mít různé délky, a mohou být měněny (mutable), což znamená, že můžete přidávat, odebírat nebo měnit jejich prvky.

V Pythonu můžete seznamy vytvořit pomocí hranatých závorek [ ] a oddělit jednotlivé prvky čárkami. Například: 

          
Muj_seznam = [1, 2, 3, 4, 5]

Vypsání seznamu

Seznam jde vypsat několika způsoby níže je ukázka pár takových vúpisů:

  • Celý list na jednou 
                  
Muj_seznam = [1, 2, 3, 4, 5]
print(Muj_seznam)
  • Jednotlive prvky seznamu 
                  
Muj_seznam = [1, 2, 3, 4, 5]
print(Muj_seznam[0])
  • Pomocí cyklu všechny prvky seznamu 
                  
Muj_seznam = [1, 2, 3, 4, 5]
for prvek in Muj_seznam:
print(prvek)

Samostaný úkol č.1

Vaším úkolem je napsat program, který bude vykreslovat dlouhou cestu z různých bloků na základě volby uživatele. Nejprve program zobrazí seznam dostupných bloků, ze kterého si uživatel může vybrat. Uživatel si poté vybere jeden z bloků ze seznamu. Program poběží v nekonečném cyklu, dokud ho uživatel neukončí stisknutím klávesy X. Výsledkem bude dlouhá cesta z vybraných bloků podle volby uživatele.

Měření času v Pythonu pomocí knihovny time

Knihovna time je něco jako stopky v počítači. Pomocí ní můžeme měřit, jak dlouho trvá, než se něco stane. Ukážeme si, jak změřit čas, jak dlouho běží cyklus, a potom tento čas převedeme na sekundy. 

            
import time

# Spustíme stopky
start = time.time()

# Cyklus, který chceme měřit
for i in range(100000):
    print(i)

# Zastavíme stopky
konec = time.time()

# Spočítáme, jak dlouho cyklus běžel
vysledny_cas = konec - start

# Převod času na sekundy
print("Cyklus běžel:", vysledny_cas, "sekund")

Vysvětlení kódu

  • time.time() - Tohle je jako zmáčknout tlačítko na stopkách.
  • start a konec - Tady uložíme čas, kdy jsme stopky spustili a kdy jsme je zastavili.
  • vysledny_cas = konec - start - Tady spočítáme, jak dlouho cyklus běžel, tím, že odčteme čas začátku od času konce.
  • print("Cyklus běžel:", vysledny_cas, "sekund") - A tady vypíšeme, jak dlouho cyklus běžel.

Samostaný úkol č.2 - Buldozer

Vašim úkolem je napsat program, který vykope určitý prostor. Velikost prostoru bude předána do programu za použití funkce input a budeto velikost x, y, z. Na konci program vypíše jak dlouho agentovy trvalo vykopat tento prostor. K tomu použijte funkci chat().

Funkce block

Nyní si přiblížíme více funkci block. Funkce block postaví na zadané souřadnice ve světě zvolený typ bloku. Používáme ji takhle: block(gold, 1, 2, 3). Jako první se nachází vždy jaký blok se použije a zbylé 3 čísla udávají souřednice kam se blok položí. POZOR - začátek tedy bod 0,0,0 se nachází tam kde stojí hráč!!!

Samostaný úkol č.3 - Buldozer s funkcí block

Vašim úkolem je upravit kod, který jste vytvořili v samostatné práci č.1. Místo ničení bloků agentem použijte funkci block. Poté porovnejte, který způsob je rychlejší.
