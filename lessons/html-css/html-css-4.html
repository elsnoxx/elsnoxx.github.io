Lekce č. 4 - Formuláře

V této lekci si ukážeme jak vkládat a vytvářet formuláře. Také si ukážeme jak tyto fotmuláře stylovat. Jako poslední věc v dnešní lekci si ukážeme vkládání multimediálních prvků na web.

Vytváření formulářů v HTML

V této části se naučíme vytvářet formuláře pomocí značek HTML a atributů.

Prvky ovládání v HTML

Tag <input> je jedním z nejčastěji používaných prvků formuláře, umožňuje vytvářet různé prvky jako textová pole, tlačítka nebo přepínače. Atribut type určuje druh prvku (např. text, radio).

Přidání formuláře na web

Pro vytvoření formuláře použijeme značku <form> a značky pro jednotlivé prvky formuláře, jako je <input> pro textová pole a tlačítka.

  • Textové pole

    Textové pole slouží k zadání textových informací, jako jsou jméno, příjmení, e-mail apod.

    <input type="text" value="text">
  • Heslové pole

    Heslové pole slouží k zadání hesla, kde se zadané znaky zobrazují jako tečky nebo hvězdičky.

    <input type="password" value="password">
  • Přepínač

    Přepínač je užitečný pro výběr jedné z několika možností, například pro výběr pohlaví.

    <input type="radio" value="radio">
  • Zaškrtávací políčko

    Zaškrtávací políčko umožňuje uživateli zaškrtnout jednu nebo více možností z několika možných.

    <input type="checkbox" value="checkbox">
  • Skryté pole

    Skryté pole se používá pro ukládání informací, které nechcete zobrazit uživateli, například identifikátorů nebo hodnot.

    <input type="hidden" value="hidden">

Tlačítka k formulářím

Aby se každý formulář poslal dále a vyplnit budou k tomu potřeba i tlačítka. Ty si ukážeme níže.

  • Tlačítko

    Tlačítko je interaktivní prvek, který lze kliknutím aktivovat. Používá se pro spouštění různých akcí nebo událostí.

    <input type="button" value="button">
  • Odeslat

    Tlačítko "Odeslat" se používá v formulářích pro odeslání dat na server k jejich zpracování.

    <input type="submit" value="submit">
  • Reset

    Tlačítko "Reset" slouží k vymazání všech hodnot v formuláři a vrácení ho do původního stavu.

    <input type="reset" value="reset">
  • Zaškrtávací políčko

    Tlačítko s výběrem umožňuje uživateli vybrat jednu možnost z několika dostupných.

    <input type="file" value= "file" >
  • Tlačítko s výběrem

    <label for="phone">Značka telefonu:</label>
    <select id="phone" name="phone">
    <option value="iPhone">iPhone</option>
    <option value="Samsung">Samsung</option>
    <option value="Huawei">Huawei</option>
    </select>

Stylování formulářů pomocí CSS

Stylování formulářů pomocí CSS je důležitou součástí vytváření atraktivních a uživatelsky přívětivých webových stránek. CSS umožňuje úpravu vzhledu různých prvků formulářů, jako jsou textová pole, tlačítka, výběrové seznamy a další. Tímto způsobem můžete přizpůsobit vzhled formulářů dle potřeb vašeho projektu, což zvyšuje uživatelskou přívětivost a estetiku vašich stránek.

Vkládání multimediálních prvků na web

Vkládání videí

Pro vložení videí na webové stránky se používá značka <video>. Tato značka umožňuje přehrávání videí přímo v prohlížeči a nabízí různé možnosti pro nastavení videa.

<video src="video.mp4" width="640" height="360" controls> Tento prohlížeč nepodporuje video značky. </video>

V tomto příkladu je src atribut určuje zdrojový soubor videa, width a height určují rozměry videa, a controls zobrazuje ovládací prvky pro přehrávání videa.

Vkládání zvuků

Pro vkládání zvuků na webové stránky se používá značka <audio>. Tato značka umožňuje přehrávání zvukových souborů přímo v prohlížeči a nabízí různé možnosti pro ovládání přehrávání.

<audio src="sound.mp3" controls autoplay loop> Tento prohlížeč nepodporuje audio značky. </audio>

V tomto příkladu je src atribut určuje zdrojový soubor zvuku, controls zobrazuje ovládací prvky pro přehrávání zvuku, autoplay spustí přehrávání zvuku automaticky a loop zajišťuje opakované přehrávání zvuku.

Vkládání animací

Pro vkládání animací na webové stránky existuje několik technik, které lze použít pouze pomocí HTML. Jednou z možností je využití animovaných obrázků typu GIF, které se automaticky přehrávají.

<img src="animation.gif" alt="Animovaný obrázek">

Tímto způsobem lze vložit animovaný obrázek GIF na webovou stránku, který se přehraje automaticky po načtení stránky.
