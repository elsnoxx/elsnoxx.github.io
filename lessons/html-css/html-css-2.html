Lekce č. 2. Seznamy, Obrázky, Tabulky

V této lekci si vysvětlíme malou část možností při tvorbě webových stránek. Touto oblastí je právě pár základních funkcí. Na vytvoření webových stránek můžeme použít strašnou spoustu nějakých technologii. V této lekci se koukmeme právě na tu nejzákladnější a to je HTML a spolu s tím i CSS.

Nejprve si představíme Seznamy, Obrázky a Tabulky. Ukážeme si vše jak se používá. Na konci si každý vyzkouší sám oři plnení úkolu.

Seznam

je to soubor nabídek, které začínají od čísla nebo značky (například tečky nebo odrážky). Umožňuje uspořádat různé informace a prezentovat je způsobem, který je pro uživatele nohem pohodlnější.

Seznamy jsou tvořené v html za pomoci kontejneru <ul> a do nej se pak vkladaji prvky s tagem <ol> pro seznam bez značeni a nebo <li> pro seznam s kolečky.

Seznam s odrážky

K vytvoreni seznamu s odrazky budem potrebovat kontejtner <ul></ul>, do nej vkladame jedtnotlive body seznamu, a každý radek se nachazi mezi tagy <li> text </li>

Obrázku

Pro přidaní obrázku nejprve potřebujeme si připravit soubor na požadovanou velikost. A poté ho přidáme za pomocí tagu <img> do našeho html dokumentu

URL adresa

Je to cesta k souboru, muže být jako odkaz na internet nebo jako odkaz na soubor ve složce

Atributy
  • Width
  • Height
  • SRC
  • ALT
  • Title

Důležite informace:

  • Čím menší obrazek tím se rychleji zobrazi
  • Velikost obrazku upravujeme po šířce neboli (widht)

Tabulky

Pro pridani tabulky je nezbytne znat tagy <table>, <tr>, <td></td>

  • <table>

    • Použiva se pro začátek a konec tabulky

  • <tr>

    • Table row neboli řádek tabulky, začína a končí řádek tabulky

  • <td>

    • Table data neboli tabulkové údaje, začíná a ukončuje buňku řádku tabulky

Doplňkové atributy
  • Border

    • používa se pro specifikaci tlouštky rámu kolem tabulky

    Ukázka kod a vyzualizace

  • Align

    • Vyrovnani obsahu cele tabulky v jednotlivých buňkách

  • Valign

    • Zarovnani vertikalní v buňce

  • Colspan

    • Kolik sloupců bude zabírat jedna buňka

  • Rowspan

    • Kolik sloupců bude zabírat jedna buňka
