Lekce 4 - python beginer Retezce Do ted jsem si hrali s cisli a pismeny. Nyni si v teto lekci ukazeme jak pracovat s retezci. Tim si ukazeme jak lepe vypisovat text efektivne a formatovat. Z pohledu pocitace je kazdy text mnozina znaku. Kazdy znak at uz male nebo velke pismeno ma svuj vlastni kod. ukazka: Alex != alex Pokud se teda zacneme v Pythone pracovat s retezcemi. Uz tim jsme schopni si vytvorit jednoduchy program. ukazka: s = "Tohle je prvni textovy retezec" print(s) Pokud ovsem bychom chteli rodelit obsah retezce na vice radku. Avsak vysledek by mel byt pouze na jednom radku pouzijeme k tomu zpetne lomitko \ ukazka: s = "Tohle je prvni textovy\ retezec" print(s) Escape sekvence Při práci s řetězci a programováním v jazyce Python se často setkáme s situacemi, kdy potřebujeme vložit speciální znaky do řetězce nebo uniknout z omezení běžného zápisu. Právě k tomu slouží escape sekvence. Tyto sekvence umožňují vkládat do řetězce znaky, které by jinak nebylo možné jednoduše vepsat. V jazyce Python jsou escape sekvence zapsány jako kombinace obráceného lomítka "\" a některých specifikovaných znaků. Kombinací tohoto speciálního znaku s jiným znakem můžeme dosáhnout vložení speciálního symbolu do řetězce. tabulka: ESCAPE SEKVENCE VÝZNAM \n novy radek \t tabulator \' apostrof \" uvozovka \\ opacne lomitko ukazka: print("Zacatek:\n\tAhoj,\n\ttimto vas zdravim\n\tPreji hezky den\nKonec") Základní operace a práce retezci V nyni se zaměříme na některé běžné operace prováděné s řetězci. Spojeni retezcu Spojeni dvou retezcu dokazeme pomocí operátoru +. ukazka: string1 = "Ahoj" string2 = "Světe" print(string1 + string2) Opakovani retezce Pokud potřebujete zduplikovat text nebo množinu znaků, dokonce i více než jednou, můžete využít násobení řetězců: ukazka: string1 = "Ahoj" print(string1 * 3) Delka retezce Délku řetězce zjistíme pomocí funkce len(), která se používá v různých kontextech. Například, pokud požádáme uživatele, aby zadal kód složený ze šesti znaků, můžeme potom zkontrolovat, zda skutečně zadal požadovaný počet znaků. ukazka: string1 = "Ahoj" print(len(string1)) Indexování v řetězci V řetězci jsou jednotlivé znaky uspořádány a můžeme k nim přistupovat pomocí jejich indexu, tedy jejich pořadového čísla. Důležité je si uvědomit, že index prvního prvku v řetězci je vždy nula. Podívejme se na příklad využití indexů: ukazka: string1 = "Ahoj Světe" print("Znak [0]: " + str(string1[0])) print("Znak [6]: " + str(string1[6])) Praktické metody pro práci s řetězci Při práci s řetězci se často setkáme s potřebou provádět různé operace, jako je nahrazování znaků, vyhledávání podřetězců atd. Pro tyto účely programovací jazyky poskytují sadu metod, které pokrývají běžné potřeby. ukazka: string1 = "vEtRníK" string1.capitalize() # Větrník string1.title() # Větrník string1.upper() # VĚTRNÍK string1.lower() # větrník string1 = "řetězec" string1.find("t") # 2 string1.find("k") # 5 string1.find("x") # -1 string1 = "řetězec" string1.rfind("t") # 4 string1.rfind("k") # 5 string1.rfind("x") # -1 string1 = "Ahoj Ahoj" string1.replace("Ahoj", "Čau") # Čau Čau string1.replace("Ahoj", "Čau", 1) # Čau Ahoj string1 = "Hello World" string1.count("l") # 3
