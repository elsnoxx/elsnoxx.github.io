Lekce č. 1. Úvod do světa tvorby webových stránek

V této lekci si vysvětlíme malou část veliké IT oblasti. Touto oblastí je právě web. Na vytvoření webových stránek můžeme použít strašnou spoustu nějakých technologii. V této lekci se koukmeme právě na tu nejzákladnější a to je HTML a spolu s tím i CSS.

Nejprve začneme s lehkou teorii a poté si uděláme naše vývojářské prostředí. Kde si bytvoříme náši první webovou stránku. Výsledkem této lekce bude tato webová stránka

Povolání ve vývoji

Designer  

Je to člověk, který vypracovává vnější vzhled stránky. Obvykle se design vytváří v grafických editorech (Photoshop, Illustrator a jiné). Specifikum práce designéra je ve vytváření příjemného a pohodlného rozhraní pro uživatele.

Vývojář  

Je to člověk zabývající se provedením designu do života. Vývojáři se mohou zabývat frontend, backendem nebo všem současně.

Tester  

Je to člověk, který testuje funkčnost stránky. Ze všech sil se snaží najít nejrůznější chyby. Mohou vývojářům jednoduše poukázat na chybu. Zkušenější testeři dokážou stanovit příčinu dané chyby.

SEO specialista  

Je to člověk, který se zabývá propagací webové stránky na internetu. Většina webových stránek se vytváří na komerční účely. Pro jejich produktivitu je nezbytné, aby měli co největší návštěvnost. Proto je pro libovolnou webovou stránku důležitým úkolem její propagace. SEO specialista se zabývá reklamou stránky, nastavením stránky pro vyhledávací systémy a ostatními podobnými úkoly.

Jak fungují stránky

Hosting

Je to služba, která poskytuje místo pro umístění informací na serveru trvale připojeném k internetu. Ve skutečnosti je hosting nějaký počítač, na který můžete umístit svou stránku a libovolný uživatel k němu bude mít přístup.

Hosting je také možné udělat pomocí svého počítači, avšak existuje několik požadavků:

  • počítač musí být pořád zapnutý
  • musí být neustály přístup k internetu
  • statická IP-adresa

Název domény

Je to jedinečný název stránky, který slouží pro přístup k hostingu. Příklad domény – site.com, site.org.

Prostředí pro vývoj

Je spousta možných prostředí, které nám umožní vývoj webových stránek. Ale já pro naši lekci vybral právě program Visual Studio Code. Hlavní důvod proč budeme používat tento program je, že jednoduše nám udělá server, který se bude automaticky aktualizovat a nemusíte ručně znova načíst celou stránku.

První webová stránka

Co to znamená HTML?

  • Hyper Text Markup Language neboli Hypertextový značkovací jazyk

Pokud jsme vše zvládly. Vytvoříme si náš první web. Souborv vytvoříme za pomocí Visual Studia Code. Ten si pojmenujeme takhle: prvniweb.html. A pokud máme vše dobře nainstalované tak stačí napsat zkratku doc . Pak bychom měli dostat nápovědu a poté ji potvrdit enterem. Výsledek by měl vypadat jak je vedle ukázáno na brázku.

Základní struktura HTML dokumentu

  • <! DOCTYPE html > 

    Určuje typ dokumentu. Je to nezbytné pro to, aby prohlížeč chápal, jak má interpretovat aktuální webovou stránku. Vždyť HTML existuje v několika verzích.

  • < html > 

    Určije začátek našeho html souboru. vněm jse uloŽeno záhlaví < head > a tělo dokumentu < body >.

  • < head > 

    Hlavička dokumentu, nazývaná také jako blok , může obsahovat text a tagy. No to, co je obsaženo v této části, není zobrazeno přímo na stránce, kromě tagu .

  • < meta > 

    Je to univerzální tag a dává celou skupinu možností, hlavně pomocí metatagů. S jejich pomocí je možné měnit kódování podstránky, přidávat klíčová slova, popis dokumentu a mnoho jiného. Na to, aby prohlížeč chápal, že má co do činění s kódováním UTF-8 (Unicode transformation format) a přidává se daný řádek.

  • < title > 

    Určuje hlavičku webové stránky, je to jeden z důležitých prvků určených k řešení mnoha úkolů. V operačním systému Windows je text hlavičky zobrazený v levém horním úhlu okna prohlížeče.

  • < style > 

    Tag určerý pro vytvoření stylů naší stranky.

  • < body > 

    Určuje hlavičku webové stránky, je to jeden z důležitých prvků určených k řešení mnoha úkolů. V operačním systému Windows je text hlavičky zobrazený v levém horním úhlu okna prohlížeče.

  • < div > 

    Sekce v dokumentu, který je stylizován pomocí CSS a lze použít pomocí JavaScriptu.

  • < h > 

    Označuje nadpis textu a má až 6 velikostí.

  • < p > 

    Označuje odstavec (paragraf) textu.

  • < a > 

    Označuje odkaz na jiné stránky nebo na text.

Základní struktura CSS

Co to znamená CSS?
  • Kaskádové styly tabulek / CSS je to jazyk stylů určujících zobrazení HTML dokumentů

Tak jelikož jsme si vytvořili náš první web. Musíme mu ještě přidat styl aby naše stránká vypadala lépe. Na to se používá CSS. Ukázka CSS souboru je na obrázky vedle. Změna stylý pomocí CSS můŽe být provedena několika způsoby. A to buď přímo v HTML souboru a nebo jako soubor, který se přiloží k HTML souboru.

Nyní si ukážeme pár atributů style
  • Změna barvy pozadí - background-color
  • Změna barvy písma - color
  • Změna stylu textu - font-family
  • Změna velikosti textu - font-size
  • Zarovnání textu - text-align
  • Odstarnění stylu u odkazu - text-decoration
  • Vytvoření místa kolem textu - margin
  • Vytvoření prostoru kolem obsahu - padding
  • Šířka a výška - width, height
  • Vytvoři box kolem textu - border-radius
