Cykly

V programování se občas stane, že potřebujeme udělat stejnou věc několikrát. Například, když chceme seřadit čísla nebo vykonat stejný úkol vícekrát. Abychom se vyhnuli psaní stejného kódu znovu a znovu, používáme cykly. Cykly nám pomáhají opakovat určité příkazy, dokud není splněna nějaká podmínka.

Co je to cyklus?

Cyklus je jako automat, který dělá určité úkoly znovu a znovu, dokud mu neřekneme, aby přestal. Představ si to jako hru, kde se hráč snaží udělat něco, dokud se mu to nedaří.

Cyklus while

První cyklus, který si ukážeme, je while. Tento cyklus se opakuje, dokud je splněna podmínka. Vypadá to takto: 

            

              
while (podmínka) {  
  // kód, který se opakuje 
}

Například, pokud chceme zobrazit číslo i, dokud je menší než 3, napíšeme:

V tomto příkladu se cyklus provede třikrát, protože i začíná na 0 a pokaždé se zvyšuje o 1.

Každé provedení cyklu se nazývá iterace. Kdybychom zapomněli napsat i++, cyklus by běžel navždy! 

            

              
var i = 0;  
while (i < 3) {  
  alert(i); // Zobrazí číslo
  i++;      // Zvyšuje i o 1
}

Cyklus for

Dalším běžným cyklem je for. Tento cyklus je jako plán, kde říkáme, co se má stát a kolikrát: 

            

              
for (začátek; podmínka; krok) {  
  // kód, který se provede 
}

Zde je příklad, který ukáže čísla od 0 do 2:

V tomto příkladu se děje:

  • Začátek: i = 0; (začínáme od nuly)
  • Podmínka: i < 3; (dokud je i menší než 3)
  • Krok: i++; (zvyšujeme i o 1 po každé iteraci)
  • Tělo: alert(i); (zobrazujeme aktuální číslo)

Celkově se cyklus for provede třikrát a zobrazí čísla 0, 1 a 2. Cykly nám pomáhají být efektivnější a šetřit čas při psaní kódu! 

            

              
for (var i = 0; i < 3; i++) {  
  alert(i); // Zobrazí číslo
}

Funkce

Někdy je potřeba opakovat stejný úkon v různých částech programu. Například při zobrazení zprávy při přihlášení uživatele, odhlášení nebo při jiné události.

Příklad funkce

    Funkce je základní stavební prvek programování v JavaScriptu. V našem příkladu je vytvořena jednoduchá funkce showMessage, která zobrazuje zprávu 'Ahoj, přátelé!'. Klíčové slovo function se používá k definování funkce, po něm následuje její název a tělo funkce, které obsahuje příkazy, které se vykonají, když je funkce zavolána.

    V následujícím kódu se funkce showMessage volá dvakrát. Tento příklad ukazuje, jak efektivně využívat funkce k opakování kódu bez nutnosti jeho duplikace. Místo toho, abychom psali stejný příkaz pro zobrazení zprávy vícekrát, jednoduše zavoláme funkci.

    Funkce nám šetří práci s opakováním stejného kódu, což zjednodušuje údržbu a čitelnost programu. 

            

              
function showMessage() {  
  alert('Ahoj, přátelé!');  
}
showMessage();
showMessage();

Lokální proměnné

    Funkce v JavaScriptu může obsahovat lokální proměnné, které jsou deklarovány uvnitř jejího těla. Tyto proměnné jsou přístupné pouze v rámci funkce, což znamená, že nejsou viditelné ani přístupné zvenčí.

    Lokální proměnné se často používají k uchování hodnot, které jsou specifické pro danou funkci, aniž by ovlivnily nebo kolidovaly s proměnnými definovanými v jiných částech kódu. To pomáhá udržovat přehlednost a snižuje riziko chyb, které mohou vzniknout, pokud byste náhodně přepsali hodnotu globální proměnné.

    V následujícím příkladu vytvoříme funkci showMessage, která využívá lokální proměnnou message k uložení textu, který se poté zobrazuje uživateli pomocí dialogového okna. 

          

            
function showMessage() {  
  var message = 'Ahoj, já jsem Jano!';  
  alert(message);  
}
showMessage();

Vnější proměnné

    Funkce v JavaScriptu může mít přístup k proměnným, které jsou definovány mimo její tělo. To umožňuje funkcím interagovat s proměnnými, které nejsou jejich parametry, což může být velmi užitečné pro sdílení dat mezi různými částmi kódu.

    Tento koncept je známý jako uzávěr (closure). Když funkce přistupuje k proměnné, která byla definována ve vnějším prostředí, může ji využít k vytvoření specifických výstupů nebo chování. Například, pokud máme proměnnou userName, která je definována mimo funkci, funkce může tuto proměnnou použít a vrátit smysluplný text.

    V následujícím příkladu vytvoříme funkci showMessage, která využívá vnější proměnnou userName k tomu, aby uživateli zobrazila personalizovanou zprávu. 

          

            
var userName = 'Jano';  
function showMessage() {  
  var message = 'Ahoj, já jsem ' + userName;  
  alert(message);  
}
showMessage();

Parametry

Při volání funkce jí můžeme předat informace prostřednictvím parametrů:

      

        
function showMessage(from, text) {  
  from = '**' + from + '**';  
  alert(from + ': ' + text);  
}
showMessage('Eva', 'Ahoj!');
showMessage('Eva', 'Jak se máš?');

Vrácení hodnoty

    Funkce může také vrátit výsledek, což je důležitá vlastnost, která umožňuje použití výsledku funkce v dalších operacích.

    Například, pokud máte funkci, která provádí nějaký výpočet, můžete použít klíčové slovo return k vrácení hodnoty, kterou chcete. To je užitečné v situacích, kdy potřebujete získat výsledek z funkce a použít ho jinde v kódu.

    V následujícím příkladu vytvoříme funkci sum, která sečte dvě čísla a vrátí jejich součet. 

              

                
function sum(a, b) {  
  return a + b;  
}
var test = sum(2, 5);  
alert(test); // 7

Samostatná práce

  • 1. Čísla od 1 do 10

    Napiš program, který pomocí cyklu for zobrazí čísla od 1 do 10 pomocí alert().

  • 2. Sčítání čísel

    Napiš funkci sumNumbers, která sečte všechna čísla od 1 do 100 a vrátí jejich součet.

  • 3. Zobrazování jmen

    Vytvoř pole s jmény a pomocí cyklu for zobraz každé jméno v samostatném okně.

  • 4. Vytvořte počítadlo

    Napiš funkci countTo, která vezme číslo jako argument a pomocí cyklu while bude počítat od 1 do daného čísla.

  • 5. Násobení čísel

    Napiš funkci multiplyTable, která vezme číslo jako argument a zobrazí jeho násobkovou tabulku (od 1 do 10).

  • 6. Hvězdičkový trojúhelník

    Vytvoř program, který pomocí cyklu for vykreslí trojúhelník z hvězdiček.
