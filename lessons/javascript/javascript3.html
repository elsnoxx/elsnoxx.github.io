Cykly

Počas písania skriptov sa môžete stretnúť s problémom, kedy sa úkon jedného typu vykonáva niekoľkokrát. Napríklad je potrebné zoradiť čísla a vykonať v každom skripte rovnaký kód. Na zjednodušenie práce boli vytvorené cykly. Tie nás zbavujú potreby opakovania jednej časti kódu niekoľkokrát.

Cyklus while

Prvý cyklus, o ktorom pôjde reč, je while. Vyzerá takto:


1. while (podmienka) {  
2.   // kód, telo cyklu 
3. }

Kým je podmienka pravdivá, kód z tela cyklu sa bude vykonávať. Napríklad, cyklus nižšie zobrazí i kým i<3:


1. var i = 0;  
2. while (i < 3) {  
3.   alert(i);  
4.   i++;  
5. }

Opakovanie cyklu sa vo vede nazýva „iterácia“. Cyklus v príklade hore robí tri iterácie. Teoreticky by sa cyklus mohol vykonávať večne, ak by v príklade vyššie nebolo i++.

Cyklus for

V práci sa najčastejšie stretávame s cyklom for:


1. for (začiatok; podmienka; krok) {  
2.   // telo cyklu  
3. }

Príklad cyklu, ktorý vykonáva alert(i) pre i od 0 do 2 vrátane (do 3):


1. var i;  
2. for (i = 0; i < 3; i++) {  
3.   alert(i);  
4. }

V príklade:

  • Začiatok: i = 0;
  • Podmienka: i < 3;
  • Krok: i++;
  • Telo: alert(i);

Funkcie

Niekedy je potrebné opakovať rovnaký úkon v samostatných častiach programu. Napríklad je potrebné zobraziť správu počas privítania návštevníka, pri odhlásení zo stránky alebo z iného dôvodu.

Príklad funkcie


1. function showMessage() {  
2.   alert('Ahojte priatelia!');  
3. }

V príklade sa ako prvé dá vidieť kľúčové slovo function, po ňom názov funkcie a telo funkcie.


1. showMessage();
2. showMessage();

Kód vyvolá správu dvakrát. Funkcie sú určené na to, aby nás zbavili nutnosti opakovať ten istý kód.

Lokálne premenné

Funkcia môže obsahovať lokálne premenné, odovzdané cez var. Tieto premenné sa dajú vidieť iba vo vnútri funkcie:


1. function showMessage() {  
2.   var message = 'Ahoj, ja som Jano!';  
3.   alert(message);  
4. }
5. showMessage();

Vonkajšie premenné

Funkcia sa v určitých prípadoch môže obrátiť k vonkajšej premennej:


1. var userName = 'Jano';  
2. function showMessage() {  
3.   var message = 'Ahoj, ja som ' + userName;  
4.   alert(message);  
5. }
6. showMessage();

Parametre

Počas vyvolania funkcie je jej možné odovzdať určitú informáciu:


1. function showMessage(from, text) {  
2.   from = '**' + from + '**';  
3.   alert(from + ': ' + text);  
4. }
5. showMessage('Eva', 'Ahoj!');
6. showMessage('Eva', 'Ako sa máš?');

Vrátenie hodnoty

Pomocou funkcie môžeme vrátiť nejaký výsledok:


1. function sum(a, b) {  
2.   return a + b;  
3. }
4. var test = sum(2, 5);  
5. alert(test); // 7
