Cykly

Při psaní skriptů se můžete setkat s problémem, kdy je potřeba určitý úkon provést několikrát. Například seřadit čísla a opakovaně provést stejný kód. K usnadnění práce byly vytvořeny cykly, které nás zbavují nutnosti psát opakující se kód.

Cyklus while

První cyklus, který si ukážeme, je while. Vypadá takto:


1. while (podmínka) {  
2.   // kód, tělo cyklu 
3. }

Dokud je podmínka splněna, bude se kód v těle cyklu vykonávat. Například cyklus níže zobrazí hodnotu i dokud je menší než 3:


1. var i = 0;  
2. while (i < 3) {  
3.   alert(i);  
4.   i++;  
5. }

Jedno opakování cyklu se nazývá iterace. Cyklus v tomto příkladu provede tři iterace. Teoreticky by mohl cyklus běžet donekonečna, kdyby v příkladu nebylo i++.

Cyklus for

Nejčastěji používaný cyklus je for:


1. for (začátek; podmínka; krok) {  
2.   // tělo cyklu  
3. }

Příklad cyklu, který zobrazí hodnotu i od 0 do 2 včetně:


1. var i;  
2. for (i = 0; i < 3; i++) {  
3.   alert(i);  
4. }

V příkladu:

  • Začátek: i = 0;
  • Podmínka: i < 3;
  • Krok: i++;
  • Tělo: alert(i);

Funkce

Někdy je potřeba opakovat stejný úkon v různých částech programu. Například při zobrazení zprávy při přihlášení uživatele, odhlášení nebo při jiné události.

Příklad funkce


1. function showMessage() {  
2.   alert('Ahoj, přátelé!');  
3. }

V příkladu vidíme klíčové slovo function, po něm název funkce a tělo funkce.


1. showMessage();
2. showMessage();

Kód vyvolá zprávu dvakrát. Funkce nám šetří práci s opakováním stejného kódu.

Lokální proměnné

Funkce může obsahovat lokální proměnné, které jsou deklarovány uvnitř funkce a jsou viditelné jen tam:


1. function showMessage() {  
2.   var message = 'Ahoj, já jsem Jano!';  
3.   alert(message);  
4. }
5. showMessage();

Vnější proměnné

Funkce může také přistupovat k vnějším proměnným:


1. var userName = 'Jano';  
2. function showMessage() {  
3.   var message = 'Ahoj, já jsem ' + userName;  
4.   alert(message);  
5. }
6. showMessage();

Parametry

Při volání funkce jí můžeme předat informace prostřednictvím parametrů:


1. function showMessage(from, text) {  
2.   from = '**' + from + '**';  
3.   alert(from + ': ' + text);  
4. }
5. showMessage('Eva', 'Ahoj!');
6. showMessage('Eva', 'Jak se máš?');

Vrácení hodnoty

Funkce může také vrátit výsledek:


1. function sum(a, b) {  
2.   return a + b;  
3. }
4. var test = sum(2, 5);  
5. alert(test); // 7
