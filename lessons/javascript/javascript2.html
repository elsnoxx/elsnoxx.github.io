Operátory

Důležitou částí učení je seznámení se s různými termíny. Než se pustíme do podrobností, je potřeba znát pár základních pojmů, abychom lépe porozuměli. Začněme s pojmem operand. Operand je to, s čím pracuje operátor. Například:

6 * 3 – Operátor násobení má dva operandy: 6 a 3.

Další pojem je unární operátor. Unární operátor pracuje jen s jedním operandem. Například unární mínus (-) změní znaménko čísla:

let x = 1;
x = -x;
alert(x); // zobrazí -1

Existuje také binární operátor, který pracuje se dvěma operandy. Například binární mínus:

let x = 3, y = 6;
alert(y - x); // zobrazí 3

Sčítání a spojování textu

Stejně jako v matematice, znak + slouží pro sčítání. Ale v JavaScriptu může také spojovat texty (tzv. řetězce). Když použijeme + s texty, spojí je dohromady:

let a = "moje" + "jméno";
alert(a); // zobrazí "mojejméno"

Když je alespoň jeden operand text, JavaScript převede i druhý operand na text:

alert('3' + 2); // zobrazí "32"
alert(2 + '3'); // zobrazí "23"

Přiřazování

Operátor = se používá pro přiřazení hodnoty do proměnné:

let x = 2 * 2 + 2;
alert(x); // zobrazí 6

Zbytek po dělení %

Operátor % vypočítá zbytek po dělení dvou čísel:

alert(5 % 2); // zobrazí 1
alert(8 % 3); // zobrazí 2
alert(6 % 3); // zobrazí 0

Funkce confirm()

Příkaz confirm() zobrazí dialogové okno s otázkou a dvěma tlačítky: OK a Zrušit. Uživatel může buď potvrdit akci tlačítkem OK (vrátí true), nebo ji zrušit (vrátí false).

Ukázka: Použití confirm()


<script>
let result = confirm("Chceš pokračovat?");
if (result) {
console.log("Uživatel potvrdil akci.");
} else {
console.log("Uživatel akci zrušil.");
}
</script>

Kliknutím na OK uživatel potvrdí, kliknutím na Zrušit akci odmítne:

V závislosti na volbě uživatele se v konzoli zobrazí odpovídající zpráva.

Podmínky

Operátor if umožňuje vykonat určitou akci jen tehdy, pokud je podmínka splněná. Zkus me se zamyslet jestli níže uvedené kody jsou stejné a nebo výsledek bude jinačí.

let rok = prompt('Kdy vznikl ITSTEP?');
if (rok != 2011) {
alert('Bohužel, špatně!');
} else {
alert('Správně!');
}
let rok = prompt('Kdy vznikl ITSTEP?');
if (rok == 2011) {
alert('Správně!');
} else {
alert('Bohužel, špatně!');
}

Porovnávací operátory a logické hodnoty

Porovnávací operátory známe i z matematiky:

SYMBOL VÝZNAM PŘÍKLAD 1 VÝSLEDEK PŘÍKLAD 2 VÝSLEDEK
> větší než 1 > 0 True 5 > 10 False
< menší než 1 < 0 True 11 < 10 False
>= větší nebo rovno 1 >= 1 True 5 >= 10 False
<= menší nebo rovno 1 <= 1 True 13 <= 10 False
== rovná se 1 == 1 True 5 == 10 False
!= nerovná se 1 != 0 True 5 != 5 False

Logické hodnoty mohou být dvě:

  • true – pravda, ano
  • false – lež, ne

Spojování podmínek pomocí logických operátorů

Logické operátory and, or a not nám umožňují spojovat více podmínek. Díky nim můžeme vytvářet složitější logické výrazy. Pojďme se podívat, jak fungují:

SYMBOL NÁZEV VÝZNAM PŘÍKLAD 1 VÝSLEDEK PŘÍKLAD 2 VÝSLEDEK
&& AND všechny podmínky musí být splněny (2 > 1) && (3 > 1) True (2 > 1) && (3 < 1) False
|| OR stačí, aby jedna podmínka byla splněna (2 > 1) || (3 < 1) True (2 < 1) || (3 < 1) False
! NOT negace výrazu !(2 > 1) False !(3 > 5) True

Podívejme se ještě na konkrétní příklad použití těchto operátorů v JavaScriptu:

let vek = 18;
let jmeno = 'Petr';
if (vek >= 18 && jmeno == 'Petr') {
alert('Vítej, Petře!');
} else {
alert('Přístup odepřen!');
}

let heslo = 'tajne';
if (vek < 18 || heslo != 'tajne') {
alert('Přístup odepřen!');
} else {
alert('Vítej zpět!');
}

Použití v JavaScriptu

Níže naleznete ukázku jednoduchých porovnávacích operátorů v JavaScriptu:

alert(2 > 1); // zobrazí true
alert(2 == 1); // zobrazí false
alert(2 != 1); // zobrazí true

Níže naleznete ukázku použití logických operátorů and, or a not v JavaScriptu:

alert(2 > 1 && 2 < 3); // zobrazí true (obě podmínky jsou pravdivé)
alert(2 == 1 || 2 != 3); // zobrazí true (druhá podmínka je pravdivá)
alert(!(2 == 1)); // zobrazí true (negace, protože 2 není rovno 1)

Escape sekvence

Při práci s řetězci a programováním v jazyce JavaScript se často setkáme s situacemi, kdy potřebujeme vložit speciální znaky do řetězce nebo uniknout z omezení běžného zápisu na webovou stránku. Právě k tomu slouží escape sekvence. Tyto sekvence umožňují vkládat do řetězce znaky, které by jinak nebylo možné jednoduše vepsat.

V jazyce JavaScript jsou escape sekvence zapsány jako kombinace obráceného lomítka "\" a některých specifikovaných znaků. Kombinací tohoto speciálního znaku s jiným znakem můžeme dosáhnout vložení speciálního symbolu do řetězce.

ESCAPE SEKVENCE VÝZNAM KÓD UKÁZKA
\n nový řádek alert( jeden řádek + "\n" + druhý řádek ) jeden řádek
druhý řádek
\t tabuláror alert( velká + "\nt" + mezera ) velká mezera
\' apostrof alert( vykreslí + "\'" + jako text ) vykreslí ' jako text
\" uvozovka alert( vykreslí + "\"" + jako text ) vykreslí " jako text
\\ opačné lomítko alert( vykreslí + "\\" + jako text ) vykreslí \ jako text

Ukázka použití:
alert("Ahoj, jmenji se Richard" + "\n"
+ "Jsem programátor a věnuji se právě vysvětlení problematiky"
+ "\"" + "escape sekvení" + "\"" +"".");

Samostatná práce

  • 1. Aritmetické operace

    Napiš program, který vypočítá součet, rozdíl, součin a podíl dvou čísel, která uživatel zadá pomocí prompt().

  • 2. Zbytek po dělení

    Vytvoř funkci, která vezme dvě čísla jako argumenty a vrátí zbytek po dělení. Zobraz výsledek pomocí alert().

  • 3. Porovnání dvou čísel

    Napiš funkci, která zjistí, které z dvou zadaných čísel je větší, menší nebo zda jsou shodná. Výsledek zobraz pomocí alert().

  • 4. Spojování textů

    Vytvoř funkci, která vezme dvě jména jako argumenty a vrátí zprávu ve formátu "Ahoj, [jméno1] a [jméno2]!"

  • 5. Podmínkový operátor

    Napiš program, který zkontroluje, zda je zadané číslo sudé nebo liché, a výsledek zobrazí uživateli.

  • 6. Uživatelský vstup a operátory

    Vytvoř program, který požádá uživatele o zadání věku a zkontroluje, zda je uživatel plnoletý (18 let a více). Zobraz příslušnou zprávu.
