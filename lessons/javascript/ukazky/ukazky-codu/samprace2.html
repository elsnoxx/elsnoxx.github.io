Úkoly pro procvičení cyklů a funkcí v JavaScriptu

1. Aritmetické operace

Napiš program, který vypočítá součet, rozdíl, součin a podíl dvou čísel, která uživatel zadá pomocí prompt().


function aritmetickeOperace() {
let cislo1 = prompt("Zadej první číslo:");
let cislo2 = prompt("Zadej druhé číslo:");

let soucet = Number(cislo1) + Number(cislo2);
let rozdil = Number(cislo1) - Number(cislo2);
let nasobeni = Number(cislo1) * Number(cislo2);
let podil = Number(cislo1) / Number(cislo2);

alert("Součet: " + soucet + "\nRozdíl: " + rozdil + "\nNásobení: " + nasobeni + "\nPodíl: " + podil);
}
2. Zbytek po dělení

Vytvoř funkci, která vezme dvě čísla jako argumenty a vrátí zbytek po dělení. Zobraz výsledek pomocí alert().


function zbytekPoDeleni(a, b) {
    return a % b;
}
alert("Zbytek po dělení: " + zbytekPoDeleni(5, 2)); // Zobrazí 1
3. Porovnání dvou čísel

Napiš funkci, která zjistí, které z dvou zadaných čísel je větší, menší nebo zda jsou shodná. Výsledek zobraz pomocí alert().


function porovnejCisla(cislo1, cislo2) {
    if (cislo1 > cislo2) {
        alert(cislo1 + " je větší než " + cislo2);
    } else if (cislo1 < cislo2) {
        alert(cislo1 + " je menší než " + cislo2);
    } else {
        alert("Obě čísla jsou stejná.");
    }
}
porovnejCisla(3, 5);
6. Uživatelský vstup a operátory

Vytvoř program, který požádá uživatele o zadání věku a zkontroluje, zda je uživatel plnoletý (18 let a více). Zobraz příslušnou zprávu.


var vek = prompt("Kolik ti je let?");
if (vek >= 18) {
alert("Jsi plnoletý.");
} else {
alert("Ještě nejsi plnoletý.");
}
5. Násobení čísel

Napiš funkci multiplyTable, která vezme číslo jako argument a zobrazí jeho násobkovou tabulku (od 1 do 10).


function multiplyTable(num) {
    for (var i = 1; i <= 10; i++) {
        alert(num + " x " + i + " = " + (num * i));
    }
}
multiplyTable(3); // Zobrazí násobkovou tabulku pro 3
6. Hvězdičkový trojúhelník

Vytvoř program, který pomocí cyklu for vykreslí trojúhelník z hvězdiček.


var rows = 5;
for (var i = 1; i <= rows; i++) {
    var stars = '';
    for (var j = 1; j <= i; j++) {
        stars += '*';
    }
    alert(stars);
}
