jQuery

Je to použitelný a jednoduchý framework Javascriptu. Dokážou se s nim vytvářet i cele stránky. Pro práci s jQuery budete muset ovládat nebo aspoň znát základy Javascriptu.

Jak pracuje jQuery ?

Na práci je potřeba si stáhnout potřebný soubor, abychom mohli tento framework použít. Do hlavičky naší html stránky přidáme tenhle text.

Odkaz pro stazeni knihovny je zde.

Začátek s jQuery

  • $(“#header”) – získanie prvku z id=”header”
  • $(“h3”) – získanie všetkých prvkov
  • $(“div#content.photo”) - získanie všetkých prvkov s triedou =”photo”, ktoré sa nachádzajú v prvku div s id=”content”
  • $(“ul li”) – získanie všetkých prvkov zo zoznamu
  • $(“ul li:first”) – získanie iba prvého prvku zo zoznamu

Seznámení s jQuery

jQuery je pohodlný, na použití jednoduchý JavaScript Framework. Kromě toho, že je jQuery multifunkční, lze s jeho pomocí dokonce vytvářet celé stránky. Pro práci s jQuery budete potřebovat základy JavaScriptu, protože tento framework používá jazyk JavaScript.

S jQuery můžete vytvářet animace, slidy, a také složité události. Protože už máme základy JavaScriptu, můžeme přejít k příkladům.

Jak funguje jQuery?

Pro práci budete potřebovat samotný framework. Můžete si jej stáhnout z oficiální stránky projektu a poté inicializovat v HTML:

<head>
  <script type="text/javascript" src="jquery.js"></script>
</head>

Jak získat prvek pomocí jQuery?

Abyste pochopili principy fungování jQuery, seznamte se se základními způsoby získávání prvků:

  • $("#header") – získání prvku s id="header"
  • $("h3") – získání všech prvků <h3>
  • $("div#content.photo") – získání všech prvků s třídou photo uvnitř div s id="content"
  • $("ul li") – získání všech prvků <li> v seznamu <ul>
  • $("ul li:first") – získání prvního prvku <li> ze seznamu <ul>

Praktické příklady

Výsuvný panel (Příklad 1)

Vytvoříme panel, který se může pohybovat nahoru a dolů po kliknutí.

<div id="panel"></div>
<p class="slide"><a href="#" class="btn-slide">Posunout Panel</a></p>

JavaScriptový kód:

$(document).ready(function(){
  $(".btn-slide").click(function(){
      $("#panel").slideToggle("slow");
  });
});

Animace (Příklad 2)

V následujícím příkladu uvidíte, jak lze pomocí několika řádků kódu vytvořit animaci čtverce, který se pohybuje, mění velikost a průhlednost:

$(document).ready(function(){
  $(".run").click(function(){
      $("#box").animate({opacity: "0.1", left: "+=400"}, 1200)
             .animate({opacity: "0.4", left: "+=160", height: "20", width: "20"}, "slow")
             .animate({opacity: "1", left: "0", height: "100", width: "100"}, "slow")
             .animate({top: "0"}, "fast")
             .slideUp()
             .slideDown("slow");
      return false;
  });
});

Animace při hover události (Příklad 3)

Tento příklad ukazuje, jak vytvořit animaci při najetí myší na prvek menu (hover). Po najetí myší se zobrazuje prvek <em>, jehož průhlednost a pozice jsou animovány.

$(document).ready(function(){
  $(".menu a").hover(function(){
      $(this).next("em").animate({opacity: "show", top: "-75"}, "slow");
  }, function(){
      $(this).next("em").animate({opacity: "hide", top: "-85"}, "fast");
  });
});

Galerie (Příklad 4)

Vytvoříme jednoduchou galerii bez nutnosti obnovovat stránku. Kliknutím na obrázek ve třídě thumbs změníme hlavní obrázek v galerii.

$(document).ready(function(){
  $("h2").append('<em></em>');
  $(".thumbs a").click(function(){
      var largePath = $(this).attr("href");
      var largeAlt = $(this).attr("title");
      $("#largeImg").attr({ src: largePath, alt: largeAlt });
      $("h2 em").html("(" + largeAlt + ")");
      return false;
  });
});
