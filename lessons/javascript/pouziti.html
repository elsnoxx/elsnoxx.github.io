Funkce v JavaScriptu

Funkce v JavaScriptu je blok kódu, který lze vykonat, když je "zavolán". Například, funkce může být zavolána, když dojde k nějaké události, jako když uživatel klikne na tlačítko.

O funkcích a událostech se dozvíte mnohem více v pozdějších kapitolách.

JavaScript v <head> nebo <body>

JavaScriptové skripty můžete umístit do jakékoliv části HTML dokumentu.

Skripty mohou být umístěny v <body> nebo v <head> části HTML stránky, nebo na obou místech.

JavaScript v <head>

V tomto příkladu je JavaScriptová funkce umístěna v <head> části HTML stránky.

Funkce je vyvolána (zavolána) při kliknutí na tlačítko:

<!DOCTYPE html>
<html>
<head>
<script>
function myFunction() {
document.getElementById("demo").innerHTML = "Odstavec změněn.";
}
</script>
</head>
<body>
<h2>Ukázka JavaScriptu v Head</h2>
<p id="demo">Odstavec</p>
<button type="button" onclick="myFunction()">Vyzkoušejte to</button>
</body>
</html>

JavaScript v <body>

V tomto příkladu je JavaScriptová funkce umístěna v <body> části HTML stránky.

Funkce je vyvolána (zavolána) při kliknutí na tlačítko:

<!DOCTYPE html>
<html>
<body>
<h2>Ukázka JavaScriptu v Body</h2>
<p id="demo">Odstavec</p>
<button type="button" onclick="myFunction()">Vyzkoušejte to</button>
<script>
function myFunction() {
document.getElementById("demo").innerHTML = "Odstavec změněn.";
}
</script>
</body>
</html>

Umístění skriptů na konec elementu <body> zlepšuje rychlost zobrazení, protože interpretace skriptu zpomaluje zobrazení.

Externí JavaScript

Skripty mohou být také umístěny v externích souborech:

Externí soubor: myScript.js

function myFunction() {
document.getElementById("demo").innerHTML = "Odstavec změněn.";
}

Externí skripty jsou praktické, když se stejný kód používá na mnoha různých webových stránkách.

JavaScriptové soubory mají příponu .js.

Pro použití externího skriptu, zadejte název skriptového souboru do atributu src (source) tagu <script>:

<script src="myScript.js"></script>

Externí skripty lze umístit do <head> nebo <body> dle potřeby.

Skript bude fungovat, jako by byl umístěn přesně tam, kde se nachází tag <script>.

Externí skripty nemohou obsahovat tagy <script>.

Výhody Externího JavaScriptu

Umístění skriptů do externích souborů má některé výhody:

  • Odděluje HTML a kód
  • Ulehčuje čtení a údržbu HTML a JavaScriptu
  • Kešované JavaScriptové soubory mohou zrychlit načítání stránky

Chcete-li přidat několik skriptových souborů na jednu stránku, použijte několik tagů <script>:

<script src="myScript1.js"></script>
<script src="myScript2.js"></script>

Externí Odkazy

Externí skript lze odkazovat třemi různými způsoby:

  • Pomocí úplné URL (úplná webová adresa)
  • Pomocí cesty k souboru (např. /js/)
  • Bez cesty

Tento příklad používá úplnou URL pro odkazování na myScript.js:

<script src="https://www.w3schools.com/js/myScript.js"></script>

Tento příklad používá cestu k souboru pro odkazování na myScript.js:

<script src="/js/myScript.js"></script>

Tento příklad používá bezcestný odkaz pro odkazování na myScript.js:

<script src="myScript.js"></script>
