Úvod do JavaScript

JavaScript byl vytvořen pro tvorbu zajímavějšího vzhledu webových stránek. Je to pohodlný programovací jazyk, který se používá pro tvorbu skriptů. Tyto skripty umožňují automatizaci různých úkonů.

Ukázka: Základní skript


<script>
    alert('Ahoj, toto je úvodní zpráva!');
</script>

Možnosti JavaScript

JavaScript má široké možnosti, zejména při práci s prohlížeči. Může měnit vzhled stránky, zpracovávat údaje o návštěvnících a komunikovat se serverem pomocí technologie AJAX.

  • Měnit styly HTML prvků
  • Skrývat nebo zobrazovat prvky
  • Zpracovávat úkony uživatele (kliknutí, pohyb kurzoru atd.)
  • Komunikace se serverem pomocí AJAX

Ukázka: Změna stylu prvku

Toto je odstavec, který změní svou barvu.

Skrývat nebo zobrazovat prvky

Tento text může být skrytý nebo zobrazený.

Zpracovávat úkony uživatele

Pohybujte kurzorem nad touto oblastí.

Co JavaScript nedokáže

JavaScript nemá přístup k operačnímu systému a nemůže číst, zapisovat ani manipulovat se soubory na pevném disku. Je omezen na práci v rámci prohlížeče pro zajištění bezpečnosti a ochrany soukromí uživatelů.

JavaScript nemůže například číst místní soubory uživatele bez jeho explicitního souhlasu, ani nemůže přímo manipulovat s hardwarem počítače. Všechny operace prováděné JavaScriptem jsou limitovány na prostředí webového prohlížeče.

Začátek práce s JavaScript

Programy v JavaScript můžeme vložit do HTML dokumentu pomocí tagu <script>.

<script> 
  alert('Ahoj!'); 
</script>

Příkaz alert() zobrazí zprávu v okně a čeká na potvrzení uživatele.

Ukázka: Zobrazení více zpráv

<script>
  alert('Zpráva 1');
  alert('Zpráva 2');
</script>

Funkce console.log()

Příkaz console.log() se používá k výpisu zpráv nebo hodnot do konzole webového prohlížeče. Je to užitečný nástroj pro ladění kódu, protože neovlivňuje uživatelské rozhraní.

Ukázka: Použití console.log()


    <script>
      console.log("Ahoj svete");
    </script>

Výstup bude zobrazen v konzoli prohlížeče (klávesová zkratka F12).

Funkce document.write()

Příkaz document.write() umožňuje zapsat text nebo HTML přímo na stránku. Tento příkaz se často používá při jednoduchých výpisech, ale jeho použití je v moderním JavaScriptu omezeno.

Ukázka: Použití document.write()


<script>
  document.write("Toto je text napsaný pomocí document.write().");
</script>

Použití class a id v CSS a JavaScriptu

Použití class v CSS a JavaScriptu

class je atribut, který může být použitý na více prvcích, aby sdílely stejné styly nebo chování. V CSS můžeme stylovat všechny prvky se stejnou třídou a v JavaScriptu můžeme s nimi pracovat hromadně.

Příklad použití třídy highlight:

    <style>
      .highlight {
        background-color: yellow;
        font-weight: bold;
      }
    </style>
    
    <p class="highlight">Toto je odstavec s třídou highlight.</p>
    <p class="highlight">Další odstavec s třídou highlight.</p>
    
    <script>
      const elements = document.getElementsByClassName('highlight');
      console.log("Počet elementů s třídou 'highlight':", elements.length);
    </script>

Použití id v CSS a JavaScriptu

id je unikátní identifikátor, který může být použitý pouze jednou na stránce. V CSS můžeme stylovat konkrétní prvek pomocí jeho id a v JavaScriptu můžeme tento prvek snadno získat a manipulovat s ním.

Příklad použití id:

    <style>
      #uniqueElement {
        color: blue;
        font-size: 20px;
      }
    </style>
    
    <p id="uniqueElement">Toto je odstavec s jedinečným ID.</p>
    
    <script>
      const element = document.getElementById('uniqueElement');
      element.style.fontWeight = 'bold';
      console.log("Text v elementu:", element.innerHTML);
    </script>

Proměnné v JavaScriptu

Proměnné jsou jako malé krabičky, do kterých můžeme uložit různé informace. Když něco uložíme do proměnné, můžeme to později použít nebo změnit. V JavaScriptu používáme k vytvoření proměnných slova var, let nebo const.

Typy proměnných

  • Čísla: Umožňují nám pracovat s čísly. Například let age = 10;
  • Řetězce: Ukládají textové informace. Například let name = "Petr";
  • Logické hodnoty: Můžou být true (pravda) nebo false (nepravda). Například let isStudent = true;
  • Pole: Umožňují uložit více hodnot do jedné proměnné. Například let colors = ["červená", "modrá", "zelená"];
  • Objekty: Umožňují uložit komplexnější data, jako například jméno a věk uživatele. Například let user = {name: "Petr", age: 30};

Příklad: Uložení zprávy do proměnné

<script> 
  var message = 'Ahoj!'; // Uložíme zprávu do proměnné
  alert(message); // Ukáže zprávu v okně
</script>

Ukázka: Definování více proměnných

<script> 
  var username = 'Petr'; // Uložíme jméno do proměnné
  var age = 30; // Uložíme věk do proměnné
  alert('Uživatel: ' + username + ', Věk: ' + age); // Ukáže uživatelské jméno a věk
</script>

Chytáky a zajímavosti

  • Proměnné jsou dynamické: Můžeme změnit hodnotu proměnné kdykoliv. Například: age = 31; změní věk uživatele na 31.
  • Nejlepší praxe: Používejte let pro proměnné, které se mohou měnit, a const pro ty, které se nemění. To pomáhá zabránit chybám.
  • Typová koherence: JavaScript je dynamicky typovaný jazyk, což znamená, že nemusíme definovat typ proměnné, když ji vytváříme.
  • Proměnné se mohou překrývat: Pokud máme proměnnou s názvem age v jedné funkci a jinou age ve druhé funkci, nebudou si navzájem překážet.

Čísla a Logické hodnoty

V JavaScriptu můžeme pracovat s čísly. Můžeme mít celá čísla (např. 1, 2, 3) nebo desetinná čísla (např. 1.5, 2.75).

Co jsou logické hodnoty?

Logické hodnoty jsou jako světlo, které může být buď zapnuté nebo vypnuté. V JavaScriptu říkáme, že logická hodnota může být buď true (pravda) nebo false (nepravda).

Ukázka: Práce s logickými hodnotami

<script>
  var isVisible = true; // Říkáme, že prvek je viditelný
  if (isVisible) { // Kontrolujeme, jestli je viditelný
    alert('Prvek je viditelný'); // Pokud je viditelný, ukáže tuto zprávu
  } else {
    alert('Prvek není viditelný'); // Pokud není viditelný, ukáže tuto zprávu
  }
</script>

Funkce document.getElementById

Funkce document.getElementById() se používá k získání přístupu k HTML elementu podle jeho ID. Tento přístup umožňuje měnit vlastnosti a obsah prvku, například měnit jeho text, styl nebo atributy.

V následující ukázce se používá document.getElementById() k měnění obrázku na stránce. Když kliknete na tlačítko "Turn on the light", změní se obrázek na ikonu zapnutého světla. Tlačítko "Turn off the light" vrátí obrázek zpět na ikonu vypnutého světla.

            
<button onclick="document.getElementById('myImage').src='/img/javascript/pic_bulbon.gif'">Turn on the light</button>
<img id="myImage" src="/img/javascript/pic_bulboff.gif" style="width:100px">
<button onclick="document.getElementById('myImage').src='/img/javascript/pic_bulboff.gif'">Turn off the light</button>

Otevření nového okna pomocí JavaScriptu

Nyní si ukážeme, jak pomocí JavaScriptu otevřít nové okno, když klikneme na obrázek. V tomto případě se otevře okno s konkrétním souborem, který je zadán ve funkci. Funkce noveOkno(soubor) využívá metodu window.open(), která umožňuje otevřít nové okno nebo záložku s daným souborem.

Funkce noveOkno má jeden vstupní parametr soubor, což je cesta k obrázku nebo souboru, který se má zobrazit. Parametry jako width a height určují velikost nového okna. V příkladu níže je obrázek, na který když kliknete, otevře se nový rám s obrázkem v definované velikosti. 

              
function noveOkno(soubor) { 
  window.open(soubor, "novyRam", "width=500,height=400") 
}

Jak to funguje?

Když kliknete na obrázek, spustí se funkce noveOkno(). Tato funkce zavolá metodu window.open(), která otevře nové okno s obrázkem. Parametr soubor odkazuje na cestu k obrázku, který chceme zobrazit. V tomto případě obrázek logo.jpg.

Onclick Event: Kliknutím na obrázek (atribut onclick) se aktivuje JavaScriptový kód, který provede otevření nového okna. Obrázek je součástí odkazu, a díky javascript:void(0) nedojde k přesměrování na jinou stránku, ale spustí se JavaScriptová funkce.

Funkce prompt()

Příkaz prompt() zobrazí vyskakovací okno, kde může uživatel zadat vstup. Výstup z tohoto okna můžeme uložit do proměnné a dále s ním pracovat. Pokud uživatel nic nezadá a potvrdí dialog, výstup bude prázdný řetězec ("").

Ukázka: Použití prompt()


<script>
  let name = prompt("Jaké je tvoje jméno?");
  console.log("Jméno uživatele:", name);
</script>

Dialogové okno se zobrazí a uživatel bude vyzván k zadání jména, které se následně zobrazí v konzoli:

Jak získat číslo z prompt()?

Výchozí hodnota z funkce prompt() je vždy text (řetězec). Pokud chcete získat číslo, musíte použít funkci Number(), která převede řetězec na číslo.

Příklad: Získání čísla z prompt()

<script>
  let age = prompt("Kolik ti je let?");
  let ageAsNumber = Number(age); // Převod na číslo
  console.log("Věk jako číslo:", ageAsNumber);
</script>

Pomocí funkce Number() můžeme převést vstup z prompt() na číslo, jak je ukázáno na obrázku.

Funkce confirm()

Příkaz confirm() zobrazí dialogové okno s otázkou a dvěma tlačítky: OK a Zrušit. Uživatel může buď potvrdit akci tlačítkem OK (vrátí true), nebo ji zrušit (vrátí false).

Ukázka: Použití confirm()


<script>
  let result = confirm("Chceš pokračovat?");
  if (result) {
    console.log("Uživatel potvrdil akci.");
  } else {
    console.log("Uživatel akci zrušil.");
  }
</script>

Kliknutím na OK uživatel potvrdí, kliknutím na Zrušit akci odmítne:

V závislosti na volbě uživatele se v konzoli zobrazí odpovídající zpráva.

Funkce alert()

Příkaz alert() zobrazí jednoduché dialogové okno s textem a tlačítkem OK. Používá se pro zobrazování zpráv nebo varování, které si uživatel musí přečíst a potvrdit.

Ukázka: Použití alert()


<script>
  alert("Toto je varovná zpráva!");
</script>

Kliknutím na OK uživatel zavře dialogové okno.

Toto je nejjednodušší způsob, jak zobrazit varování nebo informaci uživateli.
