Lekcia 5. Vytvorenie slideru v JavaScript.

Vytvorenie jednoduchej slide-show

Zaujímavým prvkom, ktorý je možné vytvoriť s pomocou JavaScriptu, je slide-show. Na jej vytvorenie bude potrebný kontajner pre slajdy a slajdy samotné.

Príklad slide-show:

  • Slide 1
  • Slide 2
  • Slide 3
  • Slide 4
  • Slide 5

Základné štýly pre slider

Do ich funkčných možností patrí možnosť:

  • vytvárať kontajner pre slajdy
  • rozmiestňovať slajdy jeden nad druhým vo vnútri kontajneru
  • regulovať správanie slajdov pri objavení a zmiznutí
  • plynulo meniť priehľadnosť slajdov na vytvorenie efektu zhasínania a objavenia

Je dôležité pamätať si, že pred prácou v CSS je potrebné zmeniť triedy a použité identifikátory, aby nedochádzalo ku konfliktom s vašimi stránkami. Poďme sa pozrieť na toto CSS:


/* Prvotné štýly */
#slides {
    position: relative;
    height: 300px;
    padding: 0px;
    margin: 0px;
    list-style-type: none;
}

.slide {
    position: absolute;
    left: 0px;
    top: 0px;
    width: 100%;
    height: 100%;
    opacity: 0;
    z-index: 1;
    transition: opacity 1s;
}

.showing {
    opacity: 1;
    z-index: 2;
}

Druhotné štýly

Poďme do našej slide-show pridať rôzne štýly, aby bola zaujímavejšia. Na demonštráciu použijeme takéto štýly:


.slide {
    font-size: 40px;
    padding: 40px;
    box-sizing: border-box;
    background: #333;
    color: #fff;
}

.slide:nth-of-type(1) {
    background: red;
}

.slide:nth-of-type(2) {
    background: orange;
}

.slide:nth-of-type(3) {
    background: green;
}

.slide:nth-of-type(4) {
    background: blue;
}

.slide:nth-of-type(5) {
    background: purple;
}

JavaScript pre slider

JavaScript pri práci so slajdami vykonáva krajne dôležitú úlohu – skrýva aktuálny slajd a zobrazuje nasledovný. Pre realizáciu tejto úlohy je potrebné zmeniť názvy tried slajdov. Kód JS:


var slides = document.querySelectorAll('#slides .slide');
var currentSlide = 0;
var slideInterval = setInterval(nextSlide, 2000);

function nextSlide() {
    slides[currentSlide].className = 'slide';
    currentSlide = (currentSlide + 1) % slides.length;
    slides[currentSlide].className = 'slide showing';
}

Pauza a ovládanie slideru

Do slideru môžeme pridať ovládacie prvky, ako napríklad tlačidlo pauzy:

JavaScript pre pauzu:


var playing = true;
var pauseButton = document.getElementById('pause');

function pauseSlideshow() {
    pauseButton.innerHTML = 'Play';
    playing = false;
    clearInterval(slideInterval);
}

function playSlideshow() {
    pauseButton.innerHTML = 'Pause';
    playing = true;
    slideInterval = setInterval(nextSlide, 2000);
}

pauseButton.onclick = function() {
    if (playing) {
        pauseSlideshow();
    } else {
        playSlideshow();
    }
};

Ovládacie tlačidlá pre nasledujúci a predchádzajúci slajd

Pridáme tlačidlá pre nasledujúci a predchádzajúci slajd:

JavaScript pre ovládanie:


function nextSlide() {
    goToSlide(currentSlide + 1);
}

function previousSlide() {
    goToSlide(currentSlide - 1);
}

function goToSlide(n) {
    slides[currentSlide].className = 'slide';
    currentSlide = (n + slides.length) % slides.length;
    slides[currentSlide].className = 'slide showing';
}

var next = document.getElementById('next');
var previous = document.getElementById('previous');

next.onclick = function() {
    pauseSlideshow();
    nextSlide();
};

previous.onclick = function() {
    pauseSlideshow();
    previousSlide();
};
