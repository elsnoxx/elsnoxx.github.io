Vytvoření jednoduché slide-show

Zajímavým prvkem, který je možné vytvořit s pomocí JavaScriptu, je slide-show. K jejímu vytvoření bude potřebný kontejner pro slidery a slidy samotné.

Základní styly pro slider

Mezi jejich funkční možnosti patří možnost:

  • vytvářet kontejner pro slidy
  • rozmisťovat slidy jeden nad druhým uvnitř kontejneru
  • regulovat chování slidů při zobrazení a zmizení
  • plynule měnit průhlednost slidů pro vytvoření efektu zhasnutí a objevení

Je důležité mít na paměti, že před prací v CSS je třeba změnit třídy a použité identifikátory, aby nedocházelo k konfliktům s vašimi stránkami. Pojďme se podívat na toto CSS:


/* Prvotní styly */
#slides {
    position: relative;
    height: 300px;
    padding: 0px;
    margin: 0px;
    list-style-type: none;
}

.slide {
    position: absolute;
    left: 0px;
    top: 0px;
    width: 100%;
    height: 100%;
    opacity: 0;
    z-index: 1;
    transition: opacity 1s;
}

.showing {
    opacity: 1;
    z-index: 2;
}

Druhotné styly

Přidejme do naší slide-show různé styly, aby byla zajímavější. Na demonstrování použijeme následující styly:


.slide {
    font-size: 40px;
    padding: 40px;
    box-sizing: border-box;
    background: #333;
    color: #fff;
}

.slide:nth-of-type(1) {
    background: red;
}

.slide:nth-of-type(2) {
    background: orange;
}

.slide:nth-of-type(3) {
    background: green;
}

.slide:nth-of-type(4) {
    background: blue;
}

.slide:nth-of-type(5) {
    background: purple;
}

JavaScript pro slider

JavaScript při práci se slidy vykonává zásadní úlohu – skrývá aktuální slid a zobrazuje následující. K realizaci této úlohy je třeba změnit názvy tříd slidů. Kód JS:


var slides = document.querySelectorAll('#slides .slide');
var currentSlide = 0;
var slideInterval = setInterval(nextSlide, 2000);

function nextSlide() {
    slides[currentSlide].className = 'slide';
    currentSlide = (currentSlide + 1) % slides.length;
    slides[currentSlide].className = 'slide showing';
}

Pauza a ovládání slideru

Do slideru můžeme přidat ovládací prvky, jako například tlačítko pauzy:

JavaScript pro pauzu:


var playing = true;
var pauseButton = document.getElementById('pause');

function pauseSlideshow() {
    pauseButton.innerHTML = 'Hraj';
    playing = false;
    clearInterval(slideInterval);
}

function playSlideshow() {
    pauseButton.innerHTML = 'Pauza';
    playing = true;
    slideInterval = setInterval(nextSlide, 2000);
}

pauseButton.onclick = function() {
    if (playing) {
        pauseSlideshow();
    } else {
        playSlideshow();
    }
};

Ovládací tlačítka pro následující a předchozí slid

Přidáme tlačítka pro následující a předchozí slid:

JavaScript pro ovládání:


function nextSlide() {
    goToSlide(currentSlide + 1);
}

function previousSlide() {
    goToSlide(currentSlide - 1);
}

function goToSlide(n) {
    slides[currentSlide].className = 'slide';
    currentSlide = (n + slides.length) % slides.length;
    slides[currentSlide].className = 'slide showing';
}

var next = document.getElementById('next');
var previous = document.getElementById('previous');

next.onclick = function() {
    pauseSlideshow();
    nextSlide();
};

previous.onclick = function() {
    pauseSlideshow();
    previousSlide();
};
