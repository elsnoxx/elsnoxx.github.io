Lekce 4. Práce s formuláři a tagy.

Co je DOM v JavaScriptu?

Představ si, že webová stránka je jako kniha. Když se díváš na stránku v prohlížeči, prohlížeč si ji načte a udělá z ní takzvaný DOM (což znamená Document Object Model nebo model objektu dokumentu). DOM je jako mapa, která ukazuje, jak je kniha (nebo webová stránka) uspořádaná.

Jak DOM funguje?

Když otevřeš webovou stránku, prohlížeč ji přetvoří do DOMu. Představ si, že každá část stránky (jako jsou nadpisy, obrázky, a texty) je jako jednotlivé kousky skládačky. Všechny tyto kousky spolu tvoří celou stránku. V DOMu je každý prvek na stránce nazýván uzel.

Jaké uzly existují?

  • Elementní uzly: To jsou věci jako <h1>, <p>, nebo <img>. Tyto uzly představují jednotlivé části stránky.
  • Textové uzly: Tyto uzly obsahují text, který vidíš na stránce. Například text v nadpisu nebo odstavci.
  • Atributové uzly: Tyto uzly mají informace o elementech, jako je barva, velikost nebo místo, kde se nacházejí.

Co můžeš s DOMem dělat?

S DOMem můžeš dělat různé věci. Například můžeš:

  • Číst obsah: Můžeš zjistit, co se nachází na stránce.
  • Měnit obsah: Můžeš upravit text nebo obrázky na stránce.
  • Přidávat nové prvky: Můžeš vytvořit nový text, obrázek nebo tlačítko a přidat ho na stránku.
  • Odstraňovat prvky: Můžeš také odstranit něco, co už na stránce není potřeba.

Příklad manipulace s DOMem

Pojďme se podívat na jednoduchý příklad. Máme tlačítko, které když klikneš, změní text na stránce. Tady je ukázka, jak to funguje: 

<h1 id="heading">Ahoj světe!</h1>
<button id="changeButton">Změnit text</button>

<script>
  // Tento kód vybere nadpis a tlačítko
  const heading = document.getElementById('heading');
  const button = document.getElementById('changeButton');

  // Přidáme akci, která se stane, když klikneš na tlačítko
  button.addEventListener('click', function() {
      // Změní text nadpisu
      heading.textContent = 'Text byl změněn!';
  });
</script>

Ahoj světe!

Práce s formulářem

Zadávání události v prohlížeči

Neodmyslitelnou součástí fungování programů je vzájemné působení jednotlivých prvků. Abychom získali reakci na aktivitu návštěvníků a na vnitřní vzájemné působení skriptů, byly vytvořeny události.

Událost je jakýsi signál prohlížeče, který říká, že došlo k určité aktivitě. Typů událostí je poměrně mnoho. Pojďme se podívat na seznam nejčastěji používaných:

Události myši:

  • click – kliknutí na prvek levým tlačítkem myši
  • contextmenu – kliknutí na prvek pravým tlačítkem myši
  • mouseover – uživatel navádí myš na určitý prvek
  • mousedown a mouseup – tlačítko myši stlačeno nebo uvolněno
  • mousemove – pohyb myši

Události ovládacích prvků:

  • submit – návštěvník odeslal formulář <form>
  • focus – návštěvník se soustředí na prvek (např. klikl na <input>).

Události klávesnice:

  • keydown – návštěvník stiskne klávesu
  • keyup – návštěvník uvolní klávesu

Přiřazení obslužných programů událostí

Jak má program reagovat, když došlo k nějakému úkonu? Právě na sledování událostí byl vytvořen obslužný program. Obslužný program je určitou funkcí, která zafunguje, jakmile dojde k nějaké události.

Existuje několik způsobů, pomocí kterých je možné přiřadit události obslužný program:

Obslužný program lze definovat přímo v návrhu, v atributu, který se nazývá on-událost. Například:

<input value="Stiskni mě" onclick="alert('Klik!')" type="button">

Příklad s funkcí countRabbits:

<!DOCTYPE HTML>
<html>
<head>
    <meta charset="utf-8">
    <script>
        function countRabbits() {
            for (var i = 1; i <= 3; i++) {
                alert("Králík číslo " + i);
            }
        }
    </script>
</head>
<body>
    <input type="button" onclick="countRabbits()" value="Spočítat králíky!" />
</body>
</html>

Získání hodnoty ze vstupního pole:

Pokud máme ve stránce formulářové pole pro uživatele, potřebujeme ho mít možnost vybrat a pak s ním manipulovat. Například:

<input id="myInput" type="text"></input>
<script>
    var inputField = document.getElementById("myInput");
    alert(inputField.value);
</script>

Práce s dokumentem

Úvod do DOM

Během práce v HTML se můžeme setkat s dynamickými změnami na stránce. Základním prvkem práce s takovými změnami je DOM (Document Object Model).

DOM představuje objektový model HTML dokumentů a slouží k dynamickým změnám prvků stránky pomocí JavaScriptu.

Vyhledávání: getElement*

DOM poskytuje metody pro vyhledávání prvků v dokumentu. Mezi hlavní metody patří:

  • document.getElementById - najde prvek podle atributu id
  • document.getElementsByTagName - najde všechny prvky s daným tagem
  • document.getElementsByName - najde všechny prvky s atributem name

Praktické příklady:

Příklad 1: Po stisknutí tlačítka změnit hodnotu.

<html>
<head>
    <meta charset="utf-8" />
    <script>
        function onButton() {
            var text = document.getElementById('text');
            text.innerHTML = "Tlačítko už bylo stisknuto";
        }
    </script>
</head>
<body>
    <form>
        <p id="text">Tlačítko nikdo nestiskl!</p>
        <input type="button" value="Stiskni mě!" onclick="onButton()" />
    </form>
</body>
</html>

Příklad 2: Po stisknutí tlačítka se změní barva na náhodnou.

<html>
<head>
    <meta charset="utf-8" />
    <script>
        function onButton() {
            var text = document.getElementById('text');
            var r = Math.floor(Math.random() * 250);
            var g = Math.floor(Math.random() * 250);
            var b = Math.floor(Math.random() * 250);
            text.style.color = 'rgb(' + r + ',' + g + ',' + b + ')';
        }
    </script>
</head>
<body>
    <form>
        <p id="text">Demo text!</p>
        <input type="button" value="Stiskni mě!" onclick="onButton()" />
    </form>
</body>
</html>

Příklad 3: Malá kalkulačka.

<html>
<head>
    <meta charset="utf-8" />
    <script>
        function onButton() {
            var aEl = document.getElementById('a');
            var bEl = document.getElementById('b');
            var cEl = document.getElementById('c');
            var a = Number(aEl.value);
            var b = Number(bEl.value);
            var c = a + b;
            cEl.value = c;
        }
    </script>
</head>
<body>
    <form>
        <p>a:<input id="a" type="number" value="0" /></p>
        <p>b:<input id="b" type="number" value="0" /></p>
        <p>a + b:<input id="c" readonly /></p>
        <input type="button" value="Stiskni mě!" onclick="onButton()" />
    </form>
</body>
</html>

Úkoly k procvičení:

  1. Vytvořte tlačítko, které při stisknutí změní barvu pozadí stránky na modrou.
  2. Vytvořte formulář, který při odeslání zobrazí hodnotu zadanou do pole.

Webová kalkulačka

Nyní si ukážeme, jak vytvořit základní kalkulačku jako je tahle. Využijeme k tomu své znalosti HTML a CSS, a nově nabyté znalosti v jazyce JavaScript.

Při vytváření této kalkulačky byl použit Boostrap pro snadné vytváření vzhledu, trocha CSS pro úpravu stylů a JavaScript, který zajistí její funkčnost.

Kalkulačka umožňuje základní matematické operace jako sčítání, odčítání, násobení a dělení, a zvládá i mazání a resetování vstupů.
