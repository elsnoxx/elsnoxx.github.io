Lekcia 4. Práca s formulármi a tagmi.

Práca s formulárom

Zadávanie udalosti v prehliadači

Neoddeliteľnou časťou fungovania programov je vzájomné pôsobenie jednotlivých prvkov. Na to, aby sme získavali reakciu na aktivitu návštevníkov a na vnútorné vzájomné pôsobenie skriptov, boli vytvorené udalosti.

Udalosť je akýsi signál prehliadača, ktorý hovorí, že došlo k určitej aktivite. Typov udalostí je pomerne veľa. Poďme si pozrieť zoznam najčastejšie používaných:

Udalosti myši:

  • click – kliknutie na prvok ľavým tlačidlom myši
  • contextmenu – kliknutie na prvok pravým tlačidlom myši
  • mouseover – používateľ navádza myš na určitý prvok
  • mousedown a mouseup – tlačidlo myši stlačili alebo pustili
  • mousemove – pohyb myši

Udalosti ovládacích prvkov:

  • submit – návštevník odoslal formulár <form>
  • focus – návštevník sa sústredí na prvok (napr. klikol na <input>).

Udalosti klávesnice:

  • keydown – návštevník stlačí klávesu
  • keyup – návštevník pustí klávesu

Priradenie obslužných programov udalostí

Ako má program reagovať, keď došlo k nejakému úkonu? Práve na sledovanie udalostí bol vytvorený obslužný program. Obslužný program je určitá funkcia, ktorá zafunguje akonáhle dôjde k nejakej udalosti.

Existuje niekoľko spôsobov, s pomocou ktorých je možné priradiť udalosti obslužný program:

Obslužný program možno zadefinovať priamo v návrhu, v atribúte, ktorý sa nazýva on-udalosť. Napríklad:

<input value="Stlač ma" onclick="alert('Klik!')" type="button">

Príklad s funkciou countRabbits:

<!DOCTYPE HTML>
<html>
<head>
    <meta charset="utf-8">
    <script>
        function countRabbits() {
            for (var i = 1; i <= 3; i++) {
                alert("Králik číslo " + i);
            }
        }
    </script>
</head>
<body>
    <input type="button" onclick="countRabbits()" value="Spočítať králikov!" />
</body>
</html>

Získanie hodnoty zo vstupného poľa:

Pre získanie hodnoty zo vstupného poľa je potrebné najskôr vytvoriť prvok:

<input type="button" value="Odpovedať" onclick="onButton()" />

Na získanie hodnoty použijeme príkaz:

var answer = document.getElementById('userText').value;

Práca s dokumentom

Úvod do DOM

Počas práce v HTML sa môžeme stretnúť s dynamickými zmenami na stránke. Základným prvkom práce s takýmito zmenami je DOM (Document Object Model).

DOM predstavuje objektový model HTML dokumentov a slúži na dynamické zmeny prvkov stránky pomocou JavaScriptu.

Vyhľadávanie: getElement*

DOM poskytuje metódy na vyhľadávanie prvkov v dokumente. Medzi hlavné metódy patrí:

  • document.getElementById - nájde prvok podľa atribútu id
  • document.getElementsByTagName - nájde všetky prvky s daným tagom
  • document.getElementsByName - nájde všetky prvky s atribútom name

Praktické príklady:

Príklad 1: Po stlačení tlačidla zmeniť hodnotu.

<html>
<head>
    <meta charset="utf-8" />
    <script>
        function onButton() {
            var text = document.getElementById('text');
            text.innerHTML = "Tlačidlo už stlačili";
        }
    </script>
</head>
<body>
    <form>
        <p id="text">Tlačidlo nikto nestlačil!</p>
        <input type="button" value="Stlač ma!" onclick="onButton()" />
    </form>
</body>
</html>

Príklad 2: Po stlačení tlačidla sa zmení farba na náhodnú.

<html>
<head>
    <meta charset="utf-8" />
    <script>
        function onButton() {
            var text = document.getElementById('text');
            var r = Math.floor(Math.random() * 250);
            var g = Math.floor(Math.random() * 250);
            var b = Math.floor(Math.random() * 250);
            text.style.color = 'rgb(' + r + ',' + g + ',' + b + ')';
        }
    </script>
</head>
<body>
    <form>
        <p id="text">Demo text!</p>
        <input type="button" value="Stlač ma!" onclick="onButton()" />
    </form>
</body>
</html>

Príklad 3: Malá kalkulačka.

<html>
<head>
    <meta charset="utf-8" />
    <script>
        function onButton() {
            var aEl = document.getElementById('a');
            var bEl = document.getElementById('b');
            var cEl = document.getElementById('c');
            var a = Number(aEl.value);
            var b = Number(bEl.value);
            var c = a + b;
            cEl.value = c;
        }
    </script>
</head>
<body>
    <form>
        <p>a:<input id="a" type="number" value="0" /></p>
        <p>b:<input id="b" type="number" value="0" /></p>
        <p>a + b:<input id="c" readonly /></p>
        <input type="button" value="Stlač ma!" onclick="onButton()" />
    </form>
</body>
</html>
